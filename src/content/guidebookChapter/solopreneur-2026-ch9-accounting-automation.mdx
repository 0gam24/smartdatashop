## 한 줄 요약

홈택스 (hometax.go.kr) 가 1인사업자 회계의 1차 도구.[^1] 매출·매입·신고·납부 통합. 회계 SaaS (영수증 자동 분류·장부 작성) 결합 시 — 1인사업자 회계 부담 80% 감소.

## 9.1 홈택스 통합 기능

[^1]:
- **사업자등록** — 비대면 등록 + 정보 변경
- **세금계산서 발급·수령** — B2B 거래 표준
- **부가가치세 신고·납부** — 1·7월
- **종합소득세 신고·납부** — 5월
- **현금영수증·신용카드영수증 자동 매입** — 매입세액 공제 기초 자료
- **연말정산** (직장인 + 1인사업자 겸업 시)

1인사업자가 홈택스만 활용해도 기본 회계 + 세무 모두 처리 가능.

## 9.2 홈택스 사용 5단계

사업자등록 후 즉시:
1. **홈택스 회원가입 + 사업자 권한 부여** (본인 ID 에 사업자 등록증 연결)
2. **공동인증서 (구 공인인증서) 발급** — 모든 신고 시 필수
3. **사업자 정보 확인** — 사업장 주소·업종·과세 유형
4. **사업용 카드·통장 등록** — 자동 매입 자료 활용
5. **첫 부가세 신고 일정 메모** (다음 1월 또는 7월)

## 9.3 회계 SaaS 의 역할

홈택스 외 회계 SaaS 도구 (시중에 다수 — 운영자 본 가이드북은 특정 제품 추천 X) 가 제공하는 핵심 기능:

- **영수증 자동 OCR** — 사진 → 텍스트 → 분류
- **카드·통장 거래 자동 연동** — 매월 거래 자동 입력
- **장부 자동 작성** — 간편장부·복식부기
- **부가세·종소세 자동 계산** — 신고 직전 점검
- **회계사·세무사 협업** — 데이터 공유

기능별 가격: 무료 (제한 기능) ~ 월 1만~10만원 (전체 기능). 본인 매출·거래 빈도에 맞게 선택.

## 9.4 1인사업자 자동화 흐름

홈택스 + 회계 SaaS 통합 시:
1. **거래 발생** — 사업용 카드 결제 or 세금계산서 발급
2. **자동 캐치** — SaaS 가 카드·홈택스 연동으로 자동 입력
3. **영수증 사진** — 매장 영수증 1초 촬영 → SaaS OCR
4. **자동 분류** — AI 가 거래를 임대료·통신·교통 등 자동 분류
5. **월말 정산** — 매월 1회 점검·수정
6. **신고 시점 (부가세·종소세)** — 자동 작성된 데이터 → 홈택스 제출

이 흐름이 정착하면 — 1인사업자가 회계에 쓰는 시간 월 5~10시간 → 1~2시간.

## 9.5 장부 작성 의무

[^2]:

| 매출 규모 | 장부 의무 |
|---|---|
| 단순경비율 대상 (매출 적음) | 영수증 정리만 (장부 X) |
| 기준경비율 대상 (중간) | 간편장부 권장 |
| 복식부기 의무 (매출 큼) | 복식부기 의무 |

복식부기 의무 매출 기준 (업종별 차이):
- 도매·소매업: 매출 3억원 이상
- 제조·건설·운수: 1억 5,000만원 이상
- 부동산임대업·서비스업: 7,500만원 이상

복식부기는 회계 SaaS 또는 세무사 위탁 권장.

## 9.6 세무사 위탁 시점

다음 도달 시 세무사 위탁 검토:
- 매출 1억원 이상 (부가세 신고 + 종소세 복잡도 ↑)
- 복식부기 의무 도달
- 직원 채용 (1인사업자 → 소상공인)
- 법인 전환 검토

세무사 비용: 월 5만~30만원 / 종소세 신고 1회 50만~200만원 (업체별 차이).

회계 SaaS + 세무사 결합 시 — SaaS 가 데이터 정리, 세무사가 신고·세제 검토.

## 9.7 매월 회계 점검 5분 체크리스트

- ☐ 사업용 카드·통장 거래 누락 없음
- ☐ 영수증 사진·OCR 완료 (회계 SaaS)
- ☐ 매월 매출·매입 합계 확인
- ☐ 부가세 예상 납부액 추산 (다음 신고 대비)
- ☐ 사업·개인 분리 위반 거래 없음

매월 5분만 점검하면 — 부가세·종소세 신고 시점에 큰 부담 없음.

## 9.8 회계 자동화의 실제 효과

1인사업자 매출 1억원 가정 — 회계 자동화 전후 비교:

**자동화 전 (수작업)**:
- 매월 영수증 정리 8시간
- 부가세 신고 직전 정리 20시간
- 종소세 신고 직전 정리 30시간
- 연 총 회계 시간 약 156시간

**자동화 후 (홈택스 + 회계 SaaS)**:
- 매월 점검 1시간
- 부가세 신고 4시간
- 종소세 신고 8시간
- 연 총 회계 시간 약 24시간

연 **132시간 절약** (월 11시간) → 본업·자기개발에 재투자 가능.

## 9.7 본인 회계 자동화 시나리오 3 종

**시나리오 A — 본인 신규 1인사업자**:
- 본인 회계 SaaS (ezBooks·삼쩜삼·캐치) 1 가입
- 본인 사업용 카드 + 통장 분리
- 본인 매월 5분 점검

**시나리오 B — 본인 매출 1억+ 1인사업자**:
- 본인 회계 SaaS + 본인 세무사 위탁 병행
- 본인 SaaS 자동 분류 + 세무사 검수
- 본인 매월 10분

**시나리오 C — 본인 매출 5억+ + 직원 채용**:
- 본인 전문 세무사 위탁 (월 30-100만원)
- 본인 회계 SaaS — 본인 일일 운영
- 본인 매월 30분

## 9.8 자주 묻는 질문 (FAQ 5)

**Q1. 본인 회계 SaaS — 본인 비용?**
A. 본인 매월 1-5만원 (사업 규모). 본인 절약 시간 가치 ↑.

**Q2. 본인 세무사 위탁 — 본인 자격?**
A. 본인 모든 1인사업자 가능. 단 비용 (월 10-100만원).

**Q3. 본인 회계 SaaS vs 세무사 — 본인 선택?**
A. 본인 매출 < 1억 → SaaS. 본인 매출 1억+ → 세무사 권장.

**Q4. 본인 홈택스 단독 — 본인 충분?**
A. 본인 신고 도구 ○. 단 본인 매월 자동 통보·정리 X → 본인 부담 ↑.

**Q5. 본인 자동화 후 본인 실수 위험?**
A. SaaS·세무사 검수 시스템. 본인 매월 1회 본인 점검 필수.

## 9.9 본인 액션 — 4 가지

본 9장 직후 본인 액션:

3. **본인 회계 SaaS 1 가입** (1주 이내)
4. **본인 매월 5분 점검** 캘린더
