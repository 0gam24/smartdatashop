## 한 줄 요약

다주택자 세금 = 종부세 + 양도세 + 취득세 누진.[^1] 본 장은 2주택 이상 본인 세금 룰 + 절세 전략 (가격 예측 0).

## 9.1 다주택자 세금 3 종

다주택자 (2주택 이상) 세금:[^1]
1. **취득세** — 매수 시 (다주택 누진)
2. **종합부동산세** — 매년 6월 1일 보유 기준 (다주택 누진)
3. **양도세** — 매도 시 (다주택 누진)

다주택 보유 → 1주택 대비 세금 ↑ — **본인 사용 vs 본인 자산 형성 의향**에 따라 보유 결정.

## 9.2 취득세 — 다주택 누진

취득세 일반 (1주택 + 무주택):
- 6억 이하 — **1%**
- 6~9억 — 1~3% (점진)
- 9억 초과 — **3%**

다주택 누진:[^2]
- **2주택** — 1주택 + 추가 매수 시 → **8%** (조정대상지역)
- **3주택 이상** — **12%**
- 비조정지역 — 다주택 1~3% 유지 (지역 따라)

본인 2주택 매수 (6억) → 취득세 약 5천만원 (vs 1주택 600만원).

## 9.3 종합부동산세 — 매년 6/1 기준

종부세 = 본인 + 배우자 보유 주택 공시가격 합산 (매년 6월 1일):[^1]
- **1주택** — 공시가 12억 초과분 과세
- **2주택 이상** — 공시가 9억 초과분 과세
- 세율: **0.5~5.0%** (초과 금액·주택 수 따라)

본인 2주택 (공시가 합 15억) → 9억 초과 6억에 0.6~1.6% 과세 → 약 360만~960만원/년.

본인 3주택 (공시가 합 20억) → 9억 초과 11억에 1.0~2.7% → 약 1,100만~2,970만원/년.

## 9.4 종부세 절세 — 명의 분산

종부세 절세:
- 본인 단독 명의 → 9억 기준 (다주택)
- 본인 + 배우자 공동 명의 → 각 1주택씩 → 12억 기준 (1주택)

본인 + 배우자 2주택 (각자 1주택) → 종부세 부담 ↓.

단, 명의 변경 시 — 취득세 (증여) 발생 — 본인 절세 효과 vs 증여세 비교.

## 9.5 양도세 — 다주택 누진

양도세 = 매도 시 (양도 차익 × 세율):[^1]
- **1주택** — 2년 거주 시 12억 이하 비과세
- **2주택** — 비과세 X — **6~45%** 누진 + 다주택 가산 10%p
- **3주택 이상** — 6~45% + 가산 20%p

본인 2주택 매도 (차익 1억) → 양도세 약 2,400만~4,500만원 (다주택 가산 포함).

본인 1주택 매도 (차익 1억, 2년 거주, 12억 이하) → 양도세 **0**.

## 9.6 1주택 비과세 — 12억 룰

1주택 비과세 조건 (8.5 복습):[^2]
- 본인 + 배우자 1주택
- 보유 2년 이상 + 거주 2년 이상
- 양도가 **12억 이하**

12억 초과 시 — 12억 초과 부분만 과세 + 장기보유특별공제 (10년+ 시 최대 80%).

## 9.7 다주택자 → 1주택자 전환 (양도세 비과세)

본인 다주택 → 본인 일부 매도 → 1주택 전환:
- 본인 2주택 → 1주택 매도 → **다주택 가산 적용** (10%p)
- 본인 3주택 → 2주택 매도 → **다주택 가산 적용** (20%p)
- 본인 다주택 → 1주택만 보유 → 잔여 1주택 비과세 (2년 거주 후)

다주택 → 1주택 전환 — 매도 순서 + 본인 잔여 1주택 거주 2년 충족 의무.

## 9.8 다주택자 — 보유 vs 매도 결정

본인 다주택 → 다음 의사결정:
- **본인 사용** (별장·자녀 거주) — 보유 유지
- **본인 자산 형성** — 본인 매도 후 다른 자산 (ETF·예금) 분산
- **본인 임대 수익** — 임대 사업자 등록 (11장 연관)
- **본인 노후 (만 65세+)** — 본인 매도 → 노후 자금 / 1주택 본인 거주 + 주택연금

본 가이드본 — 다주택 보유 / 매도 추천 X. **본인 자금·생활 의사결정**.

## 9.9 임대 사업자 등록 — 다주택 세금 절감

임대 사업자 등록:[^1]
- 본인 임대 주택 2채 이상 → 임대 사업자 등록 가능
- 종부세 합산 배제 가능 (등록 임대주택)
- 양도세 일부 감면
- 임대료 5% 룰 (11장)

임대 사업자 등록 — 본인 임대 의향 + 5년 이상 보유 의향 있을 때 권장.

본인 단기 매도 의향 → 임대 사업자 등록 효익 약함.

## 9.10 본인 다주택 — 액션 (3 시나리오)

**시나리오 A** — 본인 2주택 (1주택 거주 + 1주택 임대):
- 종부세 부담 — 매년 360만~960만원
- 본인 임대 수익 vs 부담 비교
- 본인 임대 사업자 등록 검토

**시나리오 B** — 본인 1주택 + 추가 매수 의향:
- 추가 매수 시 — 취득세 8% (조정지구)
- 종부세 추가 부담 매년
- 본인 의도 = 본인 사용 → 매수 OK
- 본인 의도 = 본인 자산 형성 → 본인 ETF·예금 대안 검토

**시나리오 C** — 본인 다주택 → 1주택 전환:
- 본인 매도 순서 — 양도세 누진 고려
- 1주택 전환 후 2년 거주 → 비과세
- 본인 자금 회수 → 다른 자산 분산

## 9.11 자주 묻는 질문 (FAQ 5)

**Q1. 본인 다주택 — 본인 종부세 부담?**
A. 본인 2주택 (15억) → 매년 360-960만원. 본인 3주택+ — 부담 ↑.

**Q2. 본인 명의 분산 — 본인 절세?**
A. 본인 + 배우자 각 1주택 → 종부세 ↓. 단 명의 변경 시 증여세.

**Q3. 본인 다주택 → 1주택 — 본인 매도 순서?**
A. 본인 양도차익 큰 주택 먼저 매도 (다주택 가산 ↑). 본인 1주택 잔여 — 2년 거주 후 비과세.

**Q4. 본인 임대 사업자 — 본인 효과?**
A. 종부세 합산 배제 + 양도세 일부 감면. 본인 5년+ 보유 의향 시.

**Q5. 본인 매수 — 본인 사용 vs 자산 형성?**
A. 본인 거주·별장 → 매수 OK. 본인 자산 형성 → ETF·예금 대안 검토.
