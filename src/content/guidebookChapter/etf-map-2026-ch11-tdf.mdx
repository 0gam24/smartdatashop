## 한 줄 요약

TDF (Target Date Fund) 는 은퇴 목표 시점에 따라 주식·채권 비중을 **자동 조정** 하는 펀드다.[^1] IRP·연금저축의 적격 투자상품. ETF 형태와 일반 펀드 형태 모두 존재.

## 11.1 TDF 구조 — Glide Path

TDF 의 핵심은 Glide Path — 은퇴 시점까지의 자산 배분 곡선이다.[^1]

- 젊을 때 (은퇴 30~40년 전): 주식 비중 80~90%, 채권 10~20%
- 중간 단계 (은퇴 10~20년 전): 주식 60%, 채권 40%
- 은퇴 임박 (은퇴 5~10년 전): 주식 30~40%, 채권 60~70%
- 은퇴 이후: 주식 20~30%, 채권 70~80%

투자자는 본인 은퇴 목표 연도 (예: 2050년 은퇴 → TDF 2050) 를 선택하면, 펀드가 자동으로 주식·채권 비중을 시간에 따라 조정.

## 11.2 TDF 상품 시리즈

운용사별로 TDF 시리즈를 발행한다 — 통상 5년 간격 (예: TDF 2030 / 2035 / 2040 / 2045 / 2050 / 2055 / 2060).[^1]

선택 기준:
- 본인 출생연도 + 약 60~65 = 은퇴 목표 연도
- 예: 1985년생 → 2050 목표 → TDF 2050

은퇴 시점이 후행 (예: 1985년생인데 TDF 2055 선택) 일수록 주식 비중 ↑, 위험 추구 ↑.

## 11.3 IRP·연금저축 적격 상품

TDF 는 IRP (개인형 퇴직연금) 와 연금저축의 적격 투자상품으로 지정됐다.[^1] 따라서 IRP·연금저축 계좌 내에서 TDF 매입 가능.

세제 혜택:
- IRP·연금저축 납입액: 연 700만원 세액공제 (총급여 5,500만원 이하 16.5%, 초과 13.2%)
- 운용 중: 매매차익·분배금 비과세 (계좌 내)
- 인출 시: 연금소득세 3.3~5.5% (만 55세 이후) 또는 기타소득세 16.5%

## 11.4 TDF ETF vs TDF 펀드

| 항목 | TDF ETF | TDF 펀드 |
|---|---|---|
| 거래 시간 | KRX 정규 시간 | 일 1회 NAV |
| 매매 단위 | 1주 | 1좌·정액 |
| 운용보수 | 0.3~0.7% | 0.5~1.5% |
| IRP·연금저축 편입 | 가능 | 가능 |

IRP·연금저축 내 적립식 매입 의도면 펀드 형태가 더 편리 (자동 매입 가능). 직접 매매·관리 의도면 ETF.

## 11.5 TDF의 한계

TDF 는 단일 상품으로 은퇴 자산 운용을 위임하는 구조다. 한계도 명확:

1. **개인 사정 무반영** — 본인의 다른 자산 (부동산·국민연금) 과 무관하게 Glide Path 진행
2. **운용보수 누적** — 30~40년 보유 시 보수 효과 누적
3. **운용사 단일 의존** — 펀드매니저·운용 전략 변화 위험
4. **국가별 자산 배분 한계** — 글로벌 분산 vs 한국 편중 운용사 차이

따라서 TDF 는 **자동 운용을 원하는 단순한 운용 도구** 로 활용 — 본인 다른 자산을 고려한 통합 의사결정은 별도.

## 11.6 TDF 선택 시 비교 변수

- 운용보수 (낮을수록 누적 효과)
- 글로벌 분산 비중 (해외 주식·채권 비율)
- Glide Path 곡선 (운용사별 차이)
- AUM (펀드 자산 규모) — 클수록 안정
- 트랙레코드 (운용 기간) — 길수록 검증 가능
