## 1. 사업 목적

1인 운영자가 한국어 데이터 콘텐츠 네트워크로 **Google AdSense 수익**을 만든다.

| 시점 | 월 수익 목표 |
|---|---|
| 1년 | 500만원 |
| 2년 | 1,000만원 |
| 3년+ | 2,000만원+ |

---

## 2. 운영 목적

1인 운영자가 **하루 1.5시간 이내**로 sustainable 운영.
나머지는 자동화·AI 협업.

- 운영자 burnout 0
- daily 부담 한계: 1.5h
- 외부 작업은 운영자만 가능 (도메인·DNS·결제 등) — 명확 분리
- AI 가 처리 가능한 모든 것은 자동화

---

## 3. 콘텐츠 목적

한국 **정부·공공기관 1차 출처** 데이터를 **가장 빠르게·정확하게·검증되게** 정리해 노출.

허용되는 1차 출처:
- .go.kr 정부 사이트 (nts·kostat·moef·molit 등)
- .or.kr 공공기관 (bok·korea 등)
- krx.co.kr 한국거래소
- kosis.kr 통계청 KOSIS
- law.go.kr 법령정보센터

금지:
- 익명 블로그·SNS
- "전문가에 따르면" 류 출처 불명
- 추측·전망·예측 (1차 발표 없으면)
- AI 훈련 데이터 환각

---

## 4. 네트워크 목적

메인 1 + 자매 4~8 = **최대 9 사이트**가 **한 덩어리로** 운영되어 합산 트래픽 극대화.

- 5 사이트 = 단일 콘텐츠 엔진의 N 출력 채널
- 같은 1차 출처 → N 가지 다른 형식·페르소나로 분기
- 메인 = Google Discover 진입 엔진
- 자매 = 페르소나별 깊이 + AdSense 수익원
- 양방향 cross-link funnel — 페이지뷰 multiplier
