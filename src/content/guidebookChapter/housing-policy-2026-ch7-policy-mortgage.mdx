## 한 줄 요약

정책 주담대 = 정부 우대 주담대 (디딤돌·보금자리·신혼) — 우대 금리 + LTV 70%.[^1] 본 장은 3 종 정책 주담대 + 본인 자격.

## 7.1 정책 주담대 — 무엇

정책 주담대 = 정부·HF 가 일반 시중은행 대출보다 우대 조건으로 제공:[^1]
- 우대 금리 (시중 -0.5~1.0%p)
- LTV 우대 (보통 70%, 일반은 50~60%)
- DSR 일부 완화
- 장기 분할 상환 (30년 표준)

3 종 — 디딤돌 / 보금자리 / 신혼부부 우대.

## 7.2 디딤돌 — 무주택 중저소득

디딤돌 — **무주택 + 중저소득 가구**:[^1]
- 대상: 무주택 세대주
- 소득: **연 6,000만원 이하** (외벌이) / 7,000만원 이하 (맞벌이)
- 자산: 본인 + 배우자 5억 이하
- 주택가: 5억 이하 (수도권 6억)
- 금리: **연 2.15~3.55%** (소득·기간 따라)
- 한도: **2.5억** (1주택자 X)
- 기간: 10·15·20·30년

신혼부부 / 다자녀 우대 → 추가 -0.2~0.7%p.

## 7.3 보금자리 — 무주택·1주택 가능

보금자리 — **무주택 + 일부 1주택**:[^1]
- 대상: 무주택 (또는 일부 1주택)
- 소득: **연 7,000만원 이하** (외벌이) / 8,500만원 이하 (맞벌이)
- 자산: 본인 + 배우자 5억 이하
- 주택가: 6억 이하
- 금리: **연 2.65~4.20%**
- 한도: **3.6억**
- 기간: 10·20·30·40년

40년 장기 분할 → 본인 월 부담 ↓ (총 이자 ↑).

## 7.4 신혼부부 우대

신혼부부 (결혼 7년 이내) 우대:[^2]
- 디딤돌 / 보금자리에 추가 우대
- 금리 **-0.2~0.7%p**
- 한도 추가 (디딤돌 3억, 보금자리 4억)
- 자산 / 소득 기준 일부 완화

본인 신혼 (만 19~39세 + 결혼 7년 이내) → 정책 주담대 1순위 활용 권장.

## 7.5 정책 주담대 vs 시중 주담대

| 항목 | 정책 주담대 | 시중 주담대 |
|---|---|---|
| 금리 | 2.15~4.20% | 4.0~5.5% |
| LTV | 최대 70% | 보통 50~60% |
| 한도 | 2.5~3.6억 | 본인 자격 |
| 기간 | 10~40년 | 10~30년 |
| 자격 | 소득·자산·주택가 제한 | 소득 ·신용 기준 |

본인 자격 충족 → **정책 주담대 1순위**.

## 7.6 신청 절차

**Step 1** — 가입 가능 여부 확인 (1~2일):
- HF (hf.go.kr) 사전 진단 도구
- 본인 소득·자산·주택가 입력 → 가입 가능 자동 표시

**Step 2** — 본인 자격 서류 준비 (1주):
- 본인 + 배우자 소득증명
- 본인 + 배우자 자산 (예금·주식·부채)
- 본인 무주택 확인서 (또는 1주택 확인)
- 주택 매매 계약서

**Step 3** — 본인 거래 은행에서 신청 (1~3주):
- 디딤돌·보금자리 — 9개 은행 (KB·신한·우리·하나·NH 등)
- 신청 → 심사 (2~3주) → 승인 → 실행

**Step 4** — 실행 + 잔금 (주택 매수 시점):
- 매도인에 잔금 지급
- 본인 명의 등기 이전

## 7.7 우대 금리 — 추가 인하 조건

본인 우대 -0.2~0.7%p 추가 가능:[^1]
- **자녀 1명** — -0.2%p
- **자녀 2명** — -0.4%p
- **자녀 3명 이상** — -0.7%p
- **청년 (만 19~34세)** — -0.2%p
- **신혼부부 (결혼 5년 이내)** — -0.2%p

본인 + 배우자 + 자녀 → 우대 누적 가능. 본인 신청 시 본인 자격 모두 검토 권장.

## 7.8 상환 — 30년 표준 + 거치 옵션

분할 상환 — 30년 표준:
- 원리금 균등 분할 (매월 동일)
- 또는 본인 일부 원금 상환 + 이자 (특수)
- 30년 후 잔액 0

거치 기간 (이자만 납입) — 1~5년 옵션:
- 본인 초기 부담 ↓
- 거치 기간 후 분할 시작
- 총 이자 ↑

신혼·청년 → 거치 1~3년 활용 권장 (초기 자녀 양육 부담 시기).

## 7.9 1주택 본인 — 디딤돌 / 보금자리 가능?

본인 이미 1주택 (예: 작은 주택) 보유 + 추가 매수 의향:
- **디딤돌** — 무주택만 (X)
- **보금자리** — 일부 1주택 가능 (5억 이하 주택 처분 약정)
- **시중 주담대** — 일반 활용 (4~5.5%)

본인 1주택 → 본인 주택 처분 + 디딤돌 / 보금자리 활용 권장 (대 주택으로 이주).

## 7.10 정책 주담대 — 본인 액션 (5 단계)

본인 정책 주담대 활용 5 단계:
1. **본인 자격 사전 확인** — HF 사전 진단 (1~2일)
2. **본인 자격 서류 준비** — 소득·자산·무주택 (1주)
3. **본인 신청 은행 선택** — 9개 은행 비교 (금리·수수료)
4. **신청 + 심사 + 승인** — 2~3주
5. **본인 매매 계약 + 잔금** — 정책 주담대 실행 + 매수 완료

본인 자녀·신혼·청년 → 우대 누적으로 본인 금리 최소화.
