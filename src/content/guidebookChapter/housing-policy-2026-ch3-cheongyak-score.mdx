## 한 줄 요약

청약 가점제 = 무주택 기간 + 부양 가족 + 납입 회수 = **최대 84점**.[^1] 본 장은 각 항목 점수 계산 + 본인 점수 추산법.

## 3.1 청약 가점제 — 84점 구조

청약 가점제 = 다음 3 항목 합산 (최대 84점):[^1]

| 항목 | 만점 | 비중 |
|---|---|---|
| 무주택 기간 | 32점 | 38% |
| 부양 가족 수 | 35점 | 42% |
| 청약통장 가입 기간 | 17점 | 20% |
| **합계** | **84점** | 100% |

서울 인기 아파트 → 가점 60~70점대 당첨선.
지방 → 가점 30~50점대 당첨선.

## 3.2 무주택 기간 — 32점

본인 + 배우자 무주택 기간 (성년 도래 후) 따라 점수 부여:[^1]

| 무주택 기간 | 점수 |
|---|---|
| 1년 미만 | 2점 |
| 1~2년 | 4점 |
| 2~3년 | 6점 |
| 3~4년 | 8점 |
| 4~5년 | 10점 |
| 5~6년 | 12점 |
| 6~7년 | 14점 |
| 7~8년 | 16점 |
| 8~9년 | 18점 |
| 9~10년 | 20점 |
| 10~11년 | 22점 |
| 11~12년 | 24점 |
| 12~13년 | 26점 |
| 13~14년 | 28점 |
| 14~15년 | 30점 |
| 15년 이상 | **32점 (만점)** |

무주택 기간 = 만 30세 이후 시점 OR 결혼 시점 OR 이혼 시점 중 가장 늦은 시점부터.

## 3.3 부양 가족 수 — 35점

본인 세대주 등본상 부양 가족 수 따라 점수:[^2]

| 부양 가족 수 | 점수 |
|---|---|
| 0명 | 5점 |
| 1명 | 10점 |
| 2명 | 15점 |
| 3명 | 20점 |
| 4명 | 25점 |
| 5명 | 30점 |
| 6명 이상 | **35점 (만점)** |

부양 가족 = 본인 배우자 + 자녀 + 부모 (60세 이상 + 동거 등본 + 무소득).

부양 가족 1명 (배우자만) → 10점.
부양 가족 3명 (배우자 + 자녀 2) → 20점.

## 3.4 청약통장 가입 기간 — 17점

청약통장 가입 후 기간:[^1]

| 가입 기간 | 점수 |
|---|---|
| 6개월 미만 | 1점 |
| 6~12개월 | 2점 |
| 1~2년 | 3점 |
| 2~3년 | 4점 |
| 3~4년 | 5점 |
| 4~5년 | 6점 |
| 5~6년 | 7점 |
| 6~7년 | 8점 |
| 7~8년 | 9점 |
| 8~9년 | 10점 |
| 9~10년 | 11점 |
| 10~11년 | 12점 |
| 11~12년 | 13점 |
| 12~13년 | 14점 |
| 13~14년 | 15점 |
| 14~15년 | 16점 |
| 15년 이상 | **17점 (만점)** |

**Tip**: 만 18세 가입 → 만 33세 만점 도달.

## 3.5 본인 점수 추산 — 3 시나리오

**시나리오 A** — 만 30세 미혼 청년:
- 무주택 0년 (만 30세 도래 시점부터 시작) → 2점
- 부양 가족 0명 → 5점
- 청약통장 5년 → 7점
- **합계 14점**

**시나리오 B** — 만 35세 신혼:
- 무주택 5년 (만 30세 도래 + 5년) → 12점
- 부양 가족 1명 (배우자) → 10점
- 청약통장 10년 → 12점
- **합계 34점**

**시나리오 C** — 만 45세 4 인 가구:
- 무주택 15년 (만 30세 + 15년) → 32점 (만점)
- 부양 가족 3명 (배우자 + 자녀 2) → 20점
- 청약통장 15년 → 17점 (만점)
- **합계 69점**

## 3.6 가점제 vs 추첨제

청약 — 가점제 + 추첨제 혼합:[^1]
- **국민주택** (LH) — 100% 가점제
- **민영주택 (85㎡ 이하)** — 가점제 75% + 추첨제 25%
- **민영주택 (85㎡ 초과)** — 가점제 50% + 추첨제 50%
- **투기과열지구·조정대상지역** — 가점제 비중 ↑

청년·신혼부부 추첨제 → 가점 낮아도 운으로 당첨 가능.

## 3.7 특별공급 (특공) — 별도 자격

특공 = 가점제 외 특별 자격으로 분양:[^2]
- **신혼부부 특공** (결혼 7년 이내) — 별도 모집
- **다자녀 특공** (자녀 3명 이상)
- **노부모 부양 특공** (60세 이상 부모 3년 이상 동거)
- **생애최초 특공** (만 30세 이상 첫 청약)
- **청년 특공** (만 19~34세 무주택, 일부 지역)

특공 — 가점제 점수 미달이라도 본인 자격 충족 시 별도 신청 권장.

## 3.8 가점 낮은 본인 — 어떻게

가점 낮은 (예: 20점대) 본인 액션:
1. **추첨제 활용** — 85㎡ 이하 25% 추첨, 가점 무관
2. **특공 활용** — 신혼부부·생애최초 특공 신청
3. **수도권 외곽** — 가점 당첨선 30~40점대 (지방)
4. **민간 분양** — LH 외 민간 분양 (가점 비중 ↓)
5. **장기 보유** — 청약통장 + 무주택 기간 누적 + 5~10년 후 재시도

## 3.9 가점 계산 — 청약홈 자동

본인 가점 정확 계산 — 마이홈 (myhome.go.kr) 또는 청약홈 자동 도구:[^2]
- 본인 정보 입력 → 가점 즉시 계산
- 매 청약 신청 시 청약홈에서 본인 가점 자동 입력
- 본인 신고 X → 자동 가점 적용

## 3.10 다음 단계 — 청약 신청

본인 점수 확인 → 다음 단계는 청약 신청 절차.

청약 신청 = 인터넷·모바일 (청약홈) → 본인 명의 + 청약통장 + 점수 → 당첨 시 계약 (4장 상세).

## 3.11 자주 묻는 질문 (FAQ 5)

**Q1. 본인 만점 84점 — 본인 가능?**
A. 본인 만 45세+ + 무주택 15년+ + 부양 6명+ + 청약통장 15년+. 본인 매우 어려움.

**Q2. 본인 점수 낮음 — 본인 청약?**
A. 본인 특공 (신혼·생애최초·청년) 또는 본인 추첨제 (85㎡ 이하 25%).

**Q3. 본인 부양가족 — 본인 사망 시 점수 ↓?**
A. ○. 본인 부양 변동 시 본인 가점 자동 재계산.

**Q4. 본인 청약통장 — 본인 다른 가족?**
A. 본인 1인 1통장. 본인 가족 각자 가입 가능.

**Q5. 본인 무주택 — 본인 부모 주택?**
A. 본인 + 배우자 무주택 기준. 본인 부모 주택 — 본인 무주택 자격 영향 X (별도 세대).

## 3.12 본인 액션 — 3 가지

본 3장 직후 본인 액션:
1. **본인 가점 자동 계산** (마이홈)
2. **본인 무주택 기간 + 청약통장 기간 누적 점검**
3. **본인 특공 자격** (신혼·청년·생애최초) 점검
