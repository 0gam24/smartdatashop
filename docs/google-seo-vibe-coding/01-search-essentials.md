# Google 검색 Essentials (검색 핵심 요소)

> Google 검색에서 웹사이트가 노출되고 좋은 실적을 내기 위한 가장 기본적인 핵심 요소
> 출처: https://developers.google.com/search/docs/essentials?hl=ko

## 개요
Google 검색 Essentials는 웹페이지·이미지·동영상 등 공개 콘텐츠가 Google 검색에 표시되도록 하는 세 가지 축으로 구성된다.

1. **기술 요구사항** — Google이 페이지를 색인하기 위해 요구하는 최소 기술 기준
2. **스팸 정책** — 위반 시 순위 하락 또는 검색 결과에서 제외될 수 있는 행위
3. **주요 권장사항** — 검색 노출과 순위에 가장 큰 영향을 미치는 모범 사례

> ⚠️ Google 검색 결과 노출 자체에는 비용이 들지 않는다. 단, 요구사항을 모두 충족해도 색인/노출이 100% 보장되지는 않는다.

---

## 1. 기술 요구사항 (Technical Requirements)
대부분의 사이트는 별다른 조치 없이도 자동으로 충족된다. 핵심 체크리스트:

- Googlebot이 페이지에 접근 가능해야 함 (robots.txt, 인증 차단 X)
- HTTP 200 OK 응답 정상 반환
- 페이지가 색인 생성 가능한 콘텐츠여야 함 (`noindex` 미설정)
- 검색결과에 표시 가능한 콘텐츠 형식 (HTML, 이미지, 동영상, PDF 등)

---

## 2. 스팸 정책 (Spam Policies) — 절대 금지 항목
- 클로킹(Cloaking): 사용자와 검색엔진에 다른 콘텐츠 제공
- 도어웨이 페이지 (특정 검색어 노출 목적의 얕은 페이지)
- 해킹된 콘텐츠, 숨겨진 텍스트/링크
- 키워드 스터핑 (유인 키워드 반복)
- 링크 스팸, 머신러닝으로 자동 생성된 저품질 콘텐츠
- 잘못된 동작(악성코드, 피싱), 사기성 행위
- 스크래핑된 콘텐츠, 무단 복제 콘텐츠

---

## 3. 주요 권장사항 (Key Best Practices)

### 콘텐츠
- 유용하고 신뢰할 수 있는 **사람 중심(People-First)** 콘텐츠 제작
- 사용자가 실제로 검색할 만한 단어를 자연스럽게 사용

### 구조
- 페이지 제목, 헤딩, 대체 텍스트, 링크 텍스트에 핵심 단어 배치
- 링크를 크롤링 가능하게 (`<a href="...">`)

### 도달
- 커뮤니티/소셜에서 사이트를 알리기
- 이미지·동영상·구조화된 데이터·JavaScript도 각각의 가이드 준수

### 노출 제어
- 노출 원치 않는 콘텐츠는 `noindex`, robots.txt, 인증 등을 통해 제어

---

## 바이브코딩 체크리스트
- [ ] robots.txt가 크롤러를 차단하지 않는가
- [ ] 모든 페이지가 200 응답을 반환하는가
- [ ] `<title>`, `meta description`, `h1`이 존재하는가
- [ ] 이미지 `alt`, 링크 `anchor text`가 의미 있는가
- [ ] `noindex` 태그가 의도된 페이지에만 적용되어 있는가
