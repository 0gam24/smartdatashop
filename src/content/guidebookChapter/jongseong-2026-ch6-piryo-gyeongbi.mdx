## 한 줄 요약

1인사업자의 사업 관련 지출은 **적격증빙 4종** (세금계산서·계산서·신용카드매출전표·현금영수증) 으로 입증해야 필요경비로 인정된다[^1]. 거래 건당 **3만원 초과** 시 적격증빙 미수취하면 **증빙불비가산세** 가 부과된다.

## 6.1 적격증빙 4종

| 증빙 | 발급 주체 | 본인 기록 의무 |
|---|---|---|
| 세금계산서 | 일반과세자 | 발급받기만 하면 자동 합산 |
| 계산서 | 면세사업자 | 동상 |
| 신용카드매출전표 | 카드사 → 국세청 자동 통보 | 사업용 신용카드 등록 시 자동 |
| 현금영수증 | 국세청 자동 누적 | 사업자등록번호로 발급받으면 자동 |

종이 영수증 / 일반 카드 영수증 (개인용) 은 적격증빙이 아니다.

## 6.2 3만원 룰 (적격증빙 의무 거래 단위)

거래 건당 **3만원 초과** (부가세 포함) 시 적격증빙 수취가 의무다[^1]. 미수취 시 적격증빙 미수취 가산세 발생.

3만원 이하 거래는 일반 영수증으로도 경비 인정 가능 (단 사업 관련성 입증 필요).

## 6.3 사업용 신용카드 — 홈택스 등록 권장

사업용으로 사용하는 신용카드를 홈택스에 등록하면 카드사가 분기마다 국세청에 자동 통보, 본인은 별도 입력 없이 신용카드매출전표 합계가 누적된다[^2]. 한도 50매까지 등록 가능[^2].

등록 절차:
1. 홈택스 [조회/발급] → [현금영수증·신용카드 매출] → [사업용 신용카드 등록]
2. 카드사·카드번호 입력 (카드 실물 필요)
3. 등록 완료 후 다음 분기부터 자동 누적

## 6.4 1인사업자가 자주 누락하는 경비 항목

- **차량유지비** — 업무용승용차 관련 경비. 단 업무전용자동차보험 미가입 / 운행기록부 미작성 시 한도 적용 (시행령 원문 직접 확인 권장).
- **통신비** — 사업용 휴대폰·인터넷·홈오피스 통신.
- **임차료** — 사무실·창고. 세금계산서 또는 임대인 동의 후 현금영수증.
- **교육훈련비** — 사업과 관련된 강의·세미나 수강료.
- **4대보험 사업주 본인 부담분** — 국민연금은 별도 소득공제, 건강보험료는 필요경비 산입 가능.
- **기업업무추진비 (구 접대비)** — 거래처와의 식사·선물. 한도 별도 (연 3,600만원 + 매출 비례 적용 — 직접 검증 권장).

## 6.5 검증 권장 항목 (운영자 직접 확인)

- **증빙불비가산세 정확한 산식** — 소득세법 제81조의6 본문 직접 확인. "미수취 금액 × 2%" 통설 검증 필요.
- **업무용승용차 한도 1,500만원** — 보험 미가입 / 운행기록부 미작성 시 적용. 시행령 원문 검증.
- **기업업무추진비 한도** — 2024년 명칭 변경 (접대비 → 기업업무추진비) 반영. 연 3,600만원 + 매출 적용 식 직접 확인.

자주 발생하는 환각 패턴:
- "가족 식사비 경비처리 OK" — **금지**. 사업 관련성 입증 불가.
- "여행 경비 모두 경비" — **금지**. 출장 입증 자료 필수.

## 6.6 본인 적격증빙 시나리오 3 종

**시나리오 A — 본인 1인 프리랜서 + 사업용 카드 1 장**:
- 본인 사업용 카드 1 장 → 홈택스 등록
- 본인 분기마다 카드사 → 국세청 자동 통보
- 본인 매출 대비 경비 80% 인정 → 본인 사업소득 ↓ → 본인 환급 ↑

**시나리오 B — 본인 점포 운영 + 임차료 + 인건비**:
- 본인 임대인에 세금계산서 발급 요청 (월 임차료)
- 본인 알바생 인건비 → 4대보험 + 세금계산서
- 본인 매출-경비 차감 후 사업소득

**시나리오 C — 본인 출장 잦은 영업직**:
- 본인 출장 1건당 식대·교통·숙박 적격증빙 수취
- 본인 거래처 식사 — 기업업무추진비 (한도 내)
- 본인 출장 일지 + 영수증 → 본인 사업 관련성 증빙

## 6.7 본인 적격증빙 — 본인 운용 체크리스트

- ☐ 본인 사업용 신용카드 1 장 + 홈택스 등록
- ☐ 본인 사업용 통장 1 개 (개인 통장 분리)
- ☐ 본인 영수증 사진 즉시 회계 SaaS 업로드 (1주 1회 정리)
- ☐ 본인 3만원 초과 거래 — 적격증빙 100% 수취
- ☐ 본인 거래처 → 본인 사업자등록번호로 세금계산서 요청
- ☐ 본인 매월 5분 — 카드·통장 거래 매칭

## 6.8 자주 묻는 질문 (FAQ 5)

**Q1. 본인 사업용 카드 사용 — 개인 비용 (가족 식사) 도 결제?**
A. **분리 권장**. 본인 사업용 카드 = 사업 거래만. 본인 개인 카드 = 개인 비용. 본인 회계 정확도 ↑.

**Q2. 본인 3만원 이하 — 적격증빙 미수취도 경비?**
A. 가능. 단 본인 사업 관련성 입증 자료 (출장 일지·고객 명단) 필요.

**Q3. 본인 차량유지비 — 100% 경비?**
A. 업무전용자동차보험 + 운행기록부 100% → 한도 X. 미가입·미작성 → 한도 1,500만원 적용 (시행령 본문 검증 권장).

**Q4. 본인 통신비 — 본인 휴대폰 1 대 전부 경비?**
A. 본인 사업용 사용 비율 (예: 70%) 만 경비. 100% X (개인 사용 분리).

**Q5. 본인 기업업무추진비 (구 접대비) — 한도?**
A. 연 3,600만원 + 매출 비례 (시행령 본문 직접 검증 권장). 본인 한도 초과 부분 — 경비 불인정.

## 6.9 본인 액션 — 4 가지

본 6장 직후 본인 액션:
1. **본인 사업용 카드 + 통장 분리** — 본인 회계 시작
2. **본인 카드 홈택스 등록** — 1주 이내
3. **본인 영수증 보관 시스템** — 회계 SaaS 또는 본인 폴더
4. **본인 매월 5분 — 거래 매칭** — 본인 매출·매입 자동 정리
