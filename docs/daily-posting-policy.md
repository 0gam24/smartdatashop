# 일일 발행 운영 방침 — 자연검색 트래픽 우선

> 2026-06-02, 멀티에이전트 자체 결정(ADR 0011) 만장일치 합의.
> 목적: AdSense "가치 부족" reject 후 재건 국면에서 *자연검색 트래픽을 가장 빠르게* 얻는 매일 발행 운영 모델.
> 위계: `docs/PURPOSE.md` > ADR > `CLAUDE.md` > 본 방침. CLAUDE.md "AdSense 거절 후 가치 재건 정책"과 정합.

## 핵심 통찰

reject 직후 신규 사이트의 *가장 빠른* 자연검색 트래픽은 **발행량이 아니라** 다음에서 나온다:
① 검색 의도 정밀 타겟 ② 시의성(Google Discover·실시간 검색) ③ 답변엔진(AEO) 인용 ④ 일관성.
양을 늘리면 scaled content 신호로 AdSense 재심사가 악화되므로, *품질·정밀 타겟·일관성*으로 트래픽을 만든다.

## 결정 (7+ 에이전트 합의)

### 1. 발행량 — 매일 1편 (증량 금지)
1편/일 상한 유지. 일관된 1편이 크롤 빈도·신뢰를 누적한다. 빠른 트래픽 욕심에 양을 늘리지 않는다.

### 2. 토픽 선정 우선순위 (검색수요 기반)
1. **시의성 제도 (기본값)** — 오늘~이번주 마감·시행·발표되는 정책. 사람들이 지금 검색하고 Google Discover 적격. (예: 종소세 마감, 6월 시행 제도, 산정특례 확대)
2. **고검색량 evergreen** — 정의·조건·신청방법 how-to. (예: "산정특례란", "청년적금 가입조건")
3. **데이터 변화** — ECOS/KOSIS 지표 piece. 데이터 저널 정체성, 주 1회 배분.

### 3. 주 1회 심층 인사이트
그 주의 시의성 토픽들을 교차 종합한 오리지널 분석 1편. 롱테일·LLM 인용 자산 + 원본 가치 앵커.

### 4. 매 글 SEO·AEO·GEO 의무
- **제목** = 실제 검색어를 앞 60자에 배치
- **tldr** = 즉답형 meta description (검색 의도에 직접 답)
- **첫 문단** = 5W1H 즉답 (AEO)
- **구조** = NewsArticle + FAQPage + HowTo LD 자동 트리거 (Q&A 2+, 번호 체크리스트 3+)
- **GEO** = 1차 출처 deep link + 모든 수치 `[^N]` 페어링
- **내부링크** 1~2 (토픽 클러스터)
- **문체** = 독자친화 완전 문장(요약체 금지) + 표 가독성 룰(셀 ≤40자·화살표 0·표 칸 각주 0)

### 5. 주간 리뷰 (운영자 5분)
Search Console 유입 쿼리 점검 → 다음 주 토픽·제목 조정. 실제로 들어오는 검색어에 맞춰 후속 글을 기획한다.

### 6. 절대 금지
증량, 템플릿 양산, 네트워크/자매 링크, 미검증 수치, 요약체.
