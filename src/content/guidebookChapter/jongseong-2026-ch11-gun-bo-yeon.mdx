## 한 줄 요약

5월 종합소득세 신고 결과는 그해 **11월부터 다음 해 10월까지 적용** 되는 건강보험 지역가입자 보험료의 산정 기초가 된다[^2]. 사업소득이 잡히면 **직장가입자 피부양자 자격 박탈** 가능성, 본인 국민연금·건강보험료 (사업주 부담분) 는 종소세 경비처리 또는 소득공제로 일부 회수 가능[^1][^3].

## 11.1 건강보험 지역가입자 — 11월 보험료 정산

국민건강보험공단은 매년 11월에 전년도 종소세 신고 자료를 반영해 지역가입자 보험료를 조정[^2]. 2026년 5월 신고분 → 2026년 11월 ~ 2027년 10월 보험료 적용.

지역가입자 보험료 산정 요소:
- **소득** — 종소세 신고된 사업소득·이자·배당·연금·기타
- **재산** — 주택·토지 (건강보험공단 자체 평가)
- **자동차** — 보유 차량

종소세 신고에서 **사업소득이 늘면 11월 보험료 인상** 으로 직결.

## 11.2 직장가입자 피부양자 박탈 — 사업소득 발생 시 즉시 위험

직장가입자의 피부양자 (배우자·부모·자녀) 가 사업자등록 후 사업소득을 신고하면 **피부양자 자격 박탈** 가능성.

### 박탈 기준 (단계적 강화 중 — 직접 검증 권장)
- 사업자등록 있는 경우 — 사업소득 0원 초과 즉시 박탈 가능성
- 사업자등록 없는 경우 — 연 사업소득 일정 금액 초과 시 박탈

피부양자 박탈 시 → **지역가입자 전환 → 별도 보험료 부과**. 가구 단위 부담 큰 변화.

### 회피 패턴
- 종소세 신고 시 사업소득이 발생할 만한 거래는 미리 확인
- 부업·프리랜서 소득이 일정 금액 초과 시 본인이 직장가입자로 전환 (직장 재취업 등)

## 11.3 국민연금 — 사업주 본인 부담분의 처리

사업주 본인이 부담하는 국민연금 보험료는 **사업 필요경비 산입 불가** [^3]. 단 **연금보험료 소득공제** (소득세법 §52) 로 별도 공제 가능. 두 효과를 혼동 금지.

## 11.4 건강보험료 — 사업주 본인 부담분은 경비 산입 가능

사업주 본인이 부담하는 건강보험료 (지역가입자 또는 직장가입자 본인 부담분) 는 **사업 필요경비 산입 가능** [^1]. 종소세 신고 시 사업 경비란에 입력.

직원이 있는 1인사업자 (실제로는 1+α 사업자) 의 경우:
- **직원 4대보험 사업주 부담분** — 사업 필요경비 산입
- **본인 4대보험** — 위 룰대로 분리 처리

## 11.5 보수외소득 — 직장가입자 추가 보험료

직장가입자라도 부업 / 임대 / 사업소득이 일정 금액 초과 시 추가 보험료 부과. 통설은 월 336만원 초과 (= 연 4,032만원 초과) 시 추가 부담이지만 **매년 변동** — 건보공단 최신 보도자료 직접 확인.

## 11.6 검증 권장 항목 (운영자 직접 확인)

- **피부양자 박탈 기준** — 2022.9 이후 단계적 강화. 건보공단 최신 보도자료.
- **보수외소득 추가 부담 임계값** — 월 336만원 등 수치는 매년 변동.
- **국민연금 / 건강보험료 경비처리 룰** — 사업주 본인 vs 직원 구분.

## 11.7 종소세 신고 전 점검 체크리스트

1. ☐ 본인 사업자등록 후 사업소득이 직장가입자 피부양자 자격에 영향 있는가?
2. ☐ 본인 사업주 국민연금·건강보험료 납입 내역을 정확히 분리해 입력했는가?
3. ☐ 직원 4대보험 사업주 부담분을 별도 경비란에 입력했는가?
4. ☐ 11월 보험료 인상 가능성을 가구 예산에 반영했는가?

## 외부 환경 — 보험료 부담의 실질 추이

건강보험료·국민연금 보험료율은 정부 결정이지만 **종합 부담의 체감**은 가계 가처분소득 대비 비율로 결정된다. CPI 가 빠르게 오르면 같은 명목 보험료라도 실질 부담이 커진다. 한국은행 ECOS 1차 자료 매일 자동 갱신.
