# smartdatashop — 목적

> 본 문서는 본 프로젝트의 **최상위 anchor**.

---

## 1. 사업 목적

1인 운영자가 한국어 데이터 단일 사이트로 **Google AdSense 수익**을 만든다.

| 시점 | 월 수익 목표 |
|---|---|
| 1년 | 500만원 |
| 2년 | 1,000만원 |
| 3년+ | 2,000만원+ |

---

## 2. 운영 목적

1인 운영자가 **하루 1.5시간 이내**로 sustainable 운영.
나머지는 자동화·AI 협업.

- 운영자 burnout 0
- daily 부담 한계: 1.5h
- 외부 작업은 운영자만 가능 (도메인·DNS·결제 등) — 명확 분리
- AI 가 처리 가능한 모든 것은 자동화

---

## 3. 콘텐츠 목적

한국 **정부·공공기관 1차 출처** 데이터를 **가장 빠르게·정확하게·검증되게** 정리해 노출.

허용되는 1차 출처:
- .go.kr 정부 사이트 (nts·kostat·moef·molit 등)
- .or.kr 공공기관 (bok·korea 등)
- krx.co.kr 한국거래소
- kosis.kr 통계청 KOSIS
- law.go.kr 법령정보센터

금지:
- 익명 블로그·SNS
- "전문가에 따르면" 류 출처 불명
- 추측·전망·예측 (1차 발표 없으면)
- AI 훈련 데이터 환각

---

## 4. 사이트 운영 원칙 (네트워크 컬렉트 폐기 — 2026-05-24)

본 사이트는 **단일 독립 사이트**로 운영한다.

- 모든 콘텐츠·SEO·수익화는 본 사이트 내부에서 완결
- 자매 사이트는 **별도 사업체**로 분리 운영 — 본 사이트 하네스·의사결정 영향 0
- 자매 cross-link·합산 트래픽·funnel 컨셉은 폐기 (ADR 0010)
- 필요 시 자매 사이트는 외부 자원으로 호출 (운영자 판단)

폐기 사유: 본 사이트 페이지에서 자매 링크 타고 들어가도 자세한 내용 부재 — 사용자 경험 저하. "한 덩어리" 컨셉이 실제로 작동하지 않음.
