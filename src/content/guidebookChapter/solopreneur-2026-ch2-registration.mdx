## 한 줄 요약

사업자등록은 사업 개시일 기준 20일 이내 의무 등록.[^1] 간이과세자 (매출 8,000만원 미만) vs 일반과세자 — 매출 규모와 매입 부가세 환급 필요성으로 결정.

## 2.1 사업자등록 의무 시점

[^1]:
- 사업 개시일로부터 **20일 이내** 사업자등록 의무
- 미등록 시: 미등록 가산세 (1년분 매출의 1% 등)
- 사업 개시 = 거래 발생일 또는 사업 준비 완료일

사업자등록 후에 발급되는 사업자등록증이 1인사업자의 모든 거래 (세금계산서·계좌 개설·관공서 입찰) 의 기본 신원.

## 2.2 간이과세자 vs 일반과세자

[^1] 기준:

| 항목 | 간이과세자 | 일반과세자 |
|---|---|---|
| 매출 기준 | 직전 연도 매출 **8,000만원 미만** | 8,000만원 이상 또는 본인 선택 |
| 부가세 신고 | 1년 1회 (1월) | 1년 2회 (1월·7월) |
| 부가세 부과율 | 매출의 0.5~3% (업종별) | 매출의 10% |
| 매입세액 공제 | 매출세액의 50~70% 한도 | 100% 공제 |
| 세금계산서 발급 | 제한 (영수증 대체 가능) | 의무 발급 |

## 2.3 간이과세자 선택 시 주의

간이과세자는 매입세액 공제가 제한되므로 — **사업 시작 시 큰 매입 비용** (예: 시설 투자, 장비 구매) 발생 예정이면 일반과세자가 유리.

예시:
- 사업 시작 시 시설 투자 3,000만원 (부가세 300만원 포함)
- 일반과세자: 매입세액 300만원 전액 환급 가능
- 간이과세자: 매입세액 일부만 공제 (업종별 50~70%)

매입 부가세 환급이 사업 초기 자금 흐름에 큰 영향.

## 2.4 간이과세자 vs 일반과세자 — 전환 시점

연 매출이 다음 기준 도달 시 강제 전환:
- 간이 → 일반: 매출 8,000만원 이상 (1년 평균)
- 일반 → 간이: 매출 8,000만원 미만 (1년 평균) — 본인 신청 시

전환 시 — 부가세 신고 방식 변경 + 매입세액 정산.

## 2.5 사업자등록 절차

홈택스 (hometax.go.kr) 비대면 신청 또는 세무서 방문:[^2]

**비대면 신청 (권장)**:
1. 홈택스 접속 → 사업자등록 신청
2. 사업장 주소·업종·과세 유형 (간이/일반) 선택
3. 신분증 + 임대차계약서 (사업장) 첨부
4. 신청 후 통상 3~7일 안에 사업자등록증 발급

업종 코드는 사업 내용에 맞게 선택 — 추후 세제 (간이 적용 업종, 부가가치율) 에 영향.

## 2.6 1인사업자 사업장 주소

사업장 주소 옵션:
- **자택 사업장** — 본인 거주지 (가장 단순)
- **공유 오피스** — 보증금·월세 부담 ↓
- **별도 임대 사업장** — 보증금·월세 부담 ↑, 인프라 ↑

자택 사업장은 사업자등록은 가능하나 — 일부 업종 (제조·음식점) 은 법적으로 거주지 사업장 불가.

## 2.7 사업자등록증 발급 후 즉시 액션

1. **사업용 통장 개설** — 사업자등록증 + 신분증
2. **사업용 신용카드 발급** — 사업 경비 자동 정산
3. **홈택스 사업자 회원 전환** — 본인 ID 에 사업자 권한 추가
4. **세금계산서 발급 도구** — 홈택스 또는 별도 SaaS

## 2.8 등록 후 첫 1개월 점검

- ☐ 사업자등록증 사본 5장 보유 (관공서 제출용)
- ☐ 사업용 통장·카드 발급 완료
- ☐ 홈택스 사업자 ID 설정 완료
- ☐ 회계 SaaS 또는 엑셀 가계부 결정 (9장 참조)
- ☐ 다음 부가세 신고 시점 확인 (3장)

## 2.7 본인 등록 시나리오 3 종

**시나리오 A — 본인 매출 3,000만원 추산 (간이과세자)**:
- 본인 간이과세자 등록 — 부가세 부담 ↓
- 본인 1월 부가세 1회 신고 (연 1회)
- 본인 매출 8천만 초과 시 일반 자동 전환

**시나리오 B — 본인 매출 1억+ 추산 (일반과세자)**:
- 본인 일반과세자 등록
- 본인 부가세 1월·7월 2회 신고
- 본인 매입세액 공제 활용

**시나리오 C — 본인 매출 미정 (간이 → 일반 전환 대비)**:
- 본인 간이 등록 시작
- 본인 매출 8천만 도달 시 자동 전환
- 본인 분기별 매출 추적

## 2.8 자주 묻는 질문 (FAQ 5)

**Q1. 본인 등록 — 본인 즉시?**
A. 본인 사업 시작 후 20일 내 의무. 늦으면 가산세.

**Q2. 본인 간이 vs 일반 — 본인 선택?**
A. 본인 매출 < 8천만 → 간이. 본인 매출 > 8천만 → 일반.

**Q3. 본인 등록 변경 (간이 → 일반) — 본인 시점?**
A. 본인 매출 8천만 초과 → 다음 해 7월 자동 전환.

**Q4. 본인 점포 없이 — 본인 등록?**
A. 가능 (재택근무·온라인 사업). 본인 주소 = 사업장.

**Q5. 본인 폐업 시 — 본인 절차?**
A. 본인 폐업 신고 — 사유 발생 후 25일 내. (10장 참조)
