## 한 줄 요약

모두채움 대상자 **717만 명**[^1] 은 국세청이 사전에 채워준 신고서를 그대로 전송만 하면 된다. 단, 정확한 환급액을 위해 **인적공제·기부금·연금계좌** 3 항목은 본인이 추가 입력해야 한다 (3장 상세). 본 챕터는 홈택스(PC) 와 손택스(모바일 앱) 의 단계별 흐름 + 본인 트러블슈팅 + FAQ 를 정리한다.

## 2.1 사전 준비물

- 공동인증서(구 공인인증서) 또는 간편인증 (네이버·카카오·PASS·삼성패스 등 6 종)
- 환급 받을 본인 명의 계좌번호
- (선택) 작년 신고서 (홈택스 → 신고/납부 → 종합소득세 → 지난 신고 내역 에서 조회 가능)

### 2.1.1 간편인증 6 종 — 본인 선택

| 인증 수단 | 가입 필요 | 본인 권장도 |
|---|---|---|
| 카카오톡 인증 | 카카오톡만 | ★★★ (가장 간편) |
| 네이버 인증 | 네이버 + 본인 인증 | ★★★ |
| 토스 인증 | 토스 앱 | ★★ |
| PASS 인증 | 통신사 PASS 앱 | ★★ |
| 삼성패스 | 갤럭시 + 삼성패스 | ★ |
| 공동인증서 | 인증서 발급 (1-2일) | ★ (구식) |

본인 신규 신고자 → 카카오톡·네이버 인증 권장 (5초 완료).

### 2.1.2 사전 자료 점검 — 신고 1주 전

본인 1주 전 자료 정리:
- ☐ 본인 매출·매입 자료 (1인사업자) 1년 합산
- ☐ 본인 + 배우자 부양가족 (소득 100만원 이하, 동거) 명단
- ☐ 본인 기부금 영수증 (종교·법정·정치)
- ☐ 본인 연금계좌 (IRP·연금저축) 납입 자료
- ☐ 본인 의료비·교육비 (인적공제 대상자별)

## 2.2 홈택스 (PC) — 6 단계

1. **로그인** — `https://hometax.go.kr` 접속 → 우상단 로그인 → 간편인증 또는 공동인증서.
2. **신고도움 자료 조회** — 메뉴 [신고/납부] → [종합소득세] → [신고도움 자료조회]. 모두채움 대상자라면 환급 예상액 표시.
3. **모두채움 신고서 열기** — [모두채움 신고/납부] 클릭 → 사전 입력된 신고서 화면 진입.
4. **3 항목 추가 입력 검증** — 인적공제 / 기부금 / 연금계좌 세액공제 칸을 클릭해 본인 데이터 확인·수정 (3장 상세).
5. **환급 계좌 입력** — 환급액이 있는 경우 본인 명의 계좌번호 입력.
6. **신고서 제출** — [신고서 제출] 클릭 → 접수번호 발급. 접수증은 PDF 로 다운로드 권장.

### 2.2.1 본인 신고 시간 — 단계별 예상

| 단계 | 본인 소요 시간 |
|---|---|
| 1. 로그인 | 30초 (간편인증) |
| 2. 신고도움 자료조회 | 1분 |
| 3. 모두채움 신고서 열기 | 30초 |
| 4. 3 항목 추가 입력 | 5-10분 (본인 자료 정리 시) |
| 5. 환급 계좌 입력 | 30초 |
| 6. 신고서 제출 | 30초 |
| **합계** | **약 8-13 분** |

본인 사전 자료 정리 완료 → 신고 자체 10분 이내.

## 2.3 손택스 (모바일 앱) — 4 단계

1. **앱 설치·로그인** — Google Play / App Store 에서 「국세청 손택스」 설치 → 간편인증 로그인.
2. **모두채움 카드** — 홈 화면에 "종합소득세 신고" 카드 노출. 모두채움 대상자는 카드 우측 상단에 "모두채움" 배지 표시.
3. **3 항목 입력 + 환급 계좌** — PC 와 동일 흐름이지만 모바일 화면 최적화. 손가락으로 항목별 토글.
4. **제출** — 하단 [신고서 제출] 버튼 → 접수번호 푸시 알림.

### 2.3.1 손택스 vs 홈택스 — 본인 선택

| 항목 | 손택스 (모바일) | 홈택스 (PC) |
|---|---|---|
| 본인 화면 | 작음 | 큼 |
| 자료 첨부 | 사진 촬영 가능 | PDF 업로드 |
| 본인 가능 시간 | 24/7 | 24/7 |
| 본인 권장 케이스 | 모두채움 그대로 | 자료 많은 1인사업자 |

본인 단순 모두채움 → 손택스 OK. 본인 복잡 신고 → 홈택스 권장.

## 2.4 신고 시 자주 발생하는 오류 5 가지

### A. 인적공제 부양가족 중복

근로자 부부가 같은 자녀를 양쪽에서 동시에 인적공제로 등록하면 **국세청이 자동 통보** 후 추징한다. 모두채움 사전 입력값이 있더라도, 부부 중 한 명만 자녀 인적공제를 받도록 본인 가족이 합의 후 한쪽이 빼야 한다.

**본인 대응**: 본인 + 배우자 합의 → 소득 높은 쪽 인적공제 (절세 효과 ↑).

### B. 작년 환급 받았던 계좌가 만료

폐기되거나 명의 변경된 계좌번호가 자동 입력되어 환급 지급 실패 사례가 매년 반복된다. **신고 전 환급 계좌 칸을 반드시 직접 확인**.

**본인 대응**: 본인 거래 은행 앱 → 계좌 활성화 상태 확인 후 입력.

### C. 모두채움 대상이 아닌데 모두채움으로 제출

모두채움 대상이 아닌 사업자가 사전 입력값만으로 신고하면 추가 입력해야 할 종합소득(임대·금융·기타) 누락이 발생한다. 모두채움 안내 메일/문자를 받지 못했다면 **자가 진단 (1장 §1.5)** 을 다시 확인.

**본인 대응**: 본인 안내문 미수령 → 본인 1장 §1.5 체크리스트 8 항목 자체 진단.

### D. 본인 IRP·연금저축 납입 누락

본인 12월 31일 직전 IRP·연금저축 추가 납입 → 국세청 자료 반영 X 가능성. 본인 직접 입력 의무.

**본인 대응**: 본인 IRP·연금저축 운용 회사 → 납입 증명서 발급 → 신고서 직접 입력.

### E. 본인 부양가족 사망·이혼 변경 미반영

본인 부양가족 사망·이혼·자녀 독립 등 → 국세청 자동 반영 X. 본인 가족관계증명서 변경 후 신고서 직접 수정 의무.

**본인 대응**: 본인 가족관계 변경 → 본인 가족관계증명서 1부 → 신고서 직접 수정.

## 2.5 모두채움 신고 — 시나리오 3 종

**시나리오 A — 직장인 + 강의료 부업**:
- 본인 회사 1곳 + 강의료 400만원 (8.8% 원천징수)
- 모두채움 안내 ○ → 회사 근로소득 + 강의료 자동 입력
- 본인 추가 입력 — IRP 납입 + 인적공제
- 본인 신고 시간 약 10분 + 환급 약 20~50만원

**시나리오 B — 간이과세 1인사업자**:
- 본인 매출 4,800만원 미만 단순경비율 사업자
- 모두채움 안내 ○ → 매출·경비 자동 입력
- 본인 추가 입력 — 부양가족 + 기부금
- 본인 신고 시간 약 15분 + 환급 또는 추가 납부

**시나리오 C — 일시 강의료 기타소득자**:
- 본인 정기 직업 X + 일시 강의·원고료 1,000만원
- 모두채움 안내 ○ → 기타소득 자동 입력 (8.8% 원천 적용)
- 본인 추가 입력 → 본인 다른 소득 없음 → 분리과세 vs 종합과세 선택
- 분리 8.8% vs 종합 6.6% 차이 → 본인 환급액 변동

## 2.6 마감 직전 트래픽 패턴

매년 5월 31일 마감을 앞둔 마지막 3 일에 트래픽이 집중된다. 국세청은 마감 직전 며칠간 **클라우드 자동 확장(autoscaling)** 을 적용하지만, 신고 시간 단축을 위해서는 **5월 셋째 주에 신고를 마치는 것이 권장** 된다.

### 2.6.1 본인 권장 신고 시점

| 시기 | 권장도 | 비고 |
|---|---|---|
| 5월 1~2주차 | ★★★ | 트래픽 적음, 자료 정리 시간 OK |
| 5월 3주차 (15-21일) | ★★ | 일부 본인 자료 자동 반영 시점 |
| 5월 4주차 (22-28일) | ★ | 트래픽 ↑ |
| 5월 마지막 주 (29-31일) | △ | 본인 신고 실패 위험 |

## 2.7 자주 묻는 질문 (FAQ 5)

**Q1. 본인 모두채움 안내문 받지 못했는데 신고해야 하나?**
A. 본인 1장 §1.5 자가 진단으로 본인 신고 대상 확인. 안내문 미수령 ≠ 신고 의무 면제.

**Q2. 본인 모두채움 그대로 vs 추가 입력 — 본인 환급액 차이는?**
A. 평균 15-30만원 (본인 가구 구성·기부·연금 납입 따라 변동). 본인 추가 입력 5분 → 환급 추가.

**Q3. 본인 손택스로 신고했는데 PDF 접수증 어떻게?**
A. 손택스 → 신고/납부 → 접수증 조회 → PDF 다운로드. 본인 이메일·SMS 전송 가능.

**Q4. 본인 5월 31일 마감 후 신고 가능?**
A. 가능. 단, 본인 가산세 (무신고·과소신고) 부담. 무신고 가산세 20% + 일별 이자.

**Q5. 본인 신고서 제출 후 수정 가능?**
A. 가능. 본인 신고서 제출 → 다시 [신고서 작성] → 수정신고 (9장 상세).

## 2.8 본인 액션 — 3 가지

본 2장 직후 본인 액션:
1. **간편인증 1 종 가입** — 카카오톡 또는 네이버 인증 (5초)
2. **본인 안내문 수령 확인** — 홈택스 → 신고도움 자료조회
3. **본인 신고 시점 확정** — 5월 1~3주차 (마감 1주 전)
