## 한 줄 요약

모두채움 대상자[^1] 가 **사전 입력값 그대로 제출하면 누락되는 3 항목** 이 있다. 본 챕터는 (1) 인적공제 부양가족 판정 (2) 기부금 영수증 종류별 공제 (3) 연금계좌 세액공제 (IRP·연금저축) 의 정확한 입력 방법 + 시나리오·FAQ 를 다룬다.

## 3.1 인적공제 — 부양가족 판정 12 가지 체크포인트

기본공제 1인당 **150만원** 의 소득공제 효과는 가구 단위에서 누락이 가장 많은 항목이다. 부양가족 1 명 추가 시 평균 환급액이 **수만~십수만 원** 늘어날 수 있다.

부양가족 판정의 4 대 요건:

1. **소득 요건** — 부양가족의 연간 소득금액이 **100만원 이하** (근로소득만 있는 경우 총급여 500만원 이하).
2. **나이 요건** — 직계존속 60세 이상, 직계비속·동거입양자 20세 이하 (장애인은 나이 요건 없음).
3. **동거 요건** — 직계존속은 별도 거주해도 됨. 형제자매·계부모는 주민등록 동거 필수.
4. **이중 등록 금지** — 같은 부양가족을 부부가 동시에 등록하면 추징 대상.

자주 누락되는 12 가지 사례:

- 군 복무 중인 자녀 (소득 없음, 동거 요건 자동 충족)
- 해외 유학 중인 자녀 (180일 이상 거주는 별도 확인)
- 장애인 등록증 보유 가족 (나이 요건 면제 + **추가공제 200만원**)
- 한부모 가구 (배우자 사망/이혼 + 미혼 자녀, **추가공제 100만원**)
- 부양 어머니의 국민연금이 100만원 이하인 경우
- 동생/처제가 학자금 외 소득 없는 경우
- 직계존속 의료비를 본인이 부담한 경우 (의료비 공제 별도)
- 부양가족 변동 (출생/사망/입양) 의 연도 내 시점
- 위탁아동 (위탁부모 거주) 의 인적공제
- 외국 국적 부양가족 (한국 거주 + 소득 요건 충족 시 가능)
- 결혼한 자녀 (배우자 통해 받지 않는 경우만)
- 본인이 단독 부양 → 분가한 형제자매와 합의 필요

### 3.1.1 부양가족 시나리오 3 종

**시나리오 A — 본인 + 자녀 2 + 노부모 2 (4 인 가구)**:
- 본인 기본공제 0 (본인은 본인 공제)
- 자녀 2 × 150만원 = 300만원
- 노부모 2 (60세+, 소득 100만원 이하) × 150만원 = 300만원
- **인적공제 합계 600만원** + 자녀 세액공제 추가

**시나리오 B — 본인 한부모 + 자녀 1**:
- 본인 기본공제 + 자녀 150만원
- 한부모 추가공제 100만원
- 자녀 1 추가공제 (만 6세 이하면 200만원)
- **합계 본인 사정 따라 250~500만원**

**시나리오 C — 부부 합의 (자녀 인적공제 분리)**:
- 본인 + 배우자 둘 다 자녀 인적공제 의향 → 추징 위험
- 본인 + 배우자 합의 → 소득 높은 쪽 인적공제 (절세 ↑)
- 본인 소득 5,000만원 + 배우자 3,000만원 → 본인이 자녀 공제

## 3.2 기부금 — 영수증 종류별 공제율

기부금은 영수증 종류에 따라 **세액공제율** 이 다르다.

| 종류 | 공제율 (1천만원 이하) | 공제율 (1천만원 초과) |
|---|---|---|
| 정치자금 (10만원 한도) | 100/110 ≈ 91% | (해당 없음) |
| 법정기부금 (이재민·국가기관) | 15% | 30% |
| 우리사주조합 기부금 | 30% | 30% |
| 지정기부금 (종교·사회복지·문화) | 15% | 30% |

입력 시 주의사항:

- **연말정산 시 누락된 기부금** 만 입력 (이미 회사에서 처리한 것은 중복 방지)
- 기부금 영수증의 **고유번호 (12자리)** 가 정확해야 함 — 손택스에서는 영수증 사진 OCR 도 지원
- 종교단체 기부금은 **지정기부금** 분류 (법정기부금 아님)
- 영수증 미보관 기부는 공제 불가

### 3.2.1 본인 기부 절세 효과 — 예시

**예시 A — 정치자금 10만원**:
- 공제율 91% → 본인 환급 약 9만원
- 1 인당 연 10만원 한도

**예시 B — 종교 (지정기부금) 100만원**:
- 공제율 15% → 본인 환급 15만원

**예시 C — 본인 적십자 (법정기부금) 50만원**:
- 공제율 15% → 본인 환급 7.5만원

본인 절세 효과 — 본인 종합 세율 (6.6~45%) 무관, 세액공제율 직접 적용.

### 3.2.2 본인 기부금 영수증 — 본인 확인

본인 영수증 점검:
- 영수증 발행 기관 = 법정·지정 단체 일치 (등록번호 12자리)
- 영수증 일자 = 본인 신고 연도 (2025년 1월 1일~12월 31일 기부 → 2026년 5월 신고)
- 영수증 본인 명의 일치

영수증 분실 시 — 기부 단체에 재발급 신청 (보통 1주 이내).

## 3.3 연금계좌 세액공제 — IRP / 연금저축

연금계좌 납입은 **세액공제율 13.2% 또는 16.5%** (총급여 5,500만원 이하는 16.5%) 의 직접 절세 효과가 있다.

| 항목 | 한도 |
|---|---|
| 연금저축 | 연 600만원 |
| IRP (개인형 퇴직연금) | 연 300만원 + 연금저축 합산 900만원 |
| 합산 한도 | **연 900만원** |

총급여 5,500만원 이하 가입자가 연 900만원 가득 납입하면 **세액공제 1,485,000원** (= 900만원 × 16.5%).

입력 시 주의사항:

- 모두채움은 **금융사 자료를 자동 반영** 하지만, 신규 가입 또는 추가 납입은 본인이 검증
- **퇴직연금 DC 형 회사 부담금** 은 세액공제 대상 아님 (회사가 부담한 것)
- 중도 인출 시 **세제 혜택 회수** + 추가 패널티 → 가입 후 만 55세까지 보유 권장
- ISA 만기 자금의 연금계좌 이전 시 추가 한도 인정 (조건 충족 시)

### 3.3.1 본인 연 한도 — 본인 액션

본인 12월 31일까지 연 한도 사용 액션:
- **본인 총급여 5,500만원 이하** — 연 900만원 가득 (환급 약 148.5만원)
- **본인 총급여 5,500만원 초과** — 연 900만원 가득 (환급 약 118.8만원)
- **본인 부담 한계** — 가족 양육·주거 부담 시 → 본인 형편 따라 300~600만원

본인 IRP·연금저축 자동이체 권장 (매월 25~75만원 = 연 300~900만원).

### 3.3.2 본인 IRP vs 연금저축 — 본인 선택

| 항목 | IRP | 연금저축 |
|---|---|---|
| 본인 가입 자격 | 근로소득자·사업소득자 | 모든 사람 |
| 본인 한도 | 연 300만원 + 연금저축 합산 900만원 | 연 600만원 |
| 본인 운용 | ETF·예금·펀드 | ETF·예금·펀드 |
| 본인 수령 시점 | 만 55세+ | 만 55세+ |

본인 둘 다 가입 → 본인 한도 최대화 (900만원 / 년).

## 3.4 환급액 차이 가시화

3 항목 모두 정확히 입력하면 모두채움 default 대비 **수십만원~수백만원** 의 환급액 차이가 발생한다. 정확한 차이는 본인 가구 상황에 따라 달라지며, **신고 후 즉시 홈택스 → [예상 환급액] 화면** 에서 확인 가능하다.

작년 통계 기준 모두채움만 그대로 제출한 신고자의 평균 추가 환급 누락액은 국세청 「종합소득세 신고 결과 분석」 보고서에서 확인할 수 있다.

### 3.4.1 본인 추가 환급 — 예시 시뮬레이션

본인 (직장인, 총급여 5,000만원, 자녀 2, 부모 1, IRP 700만원 납입, 종교 기부 50만원):
- 인적공제 추가 — 자녀 2 + 부모 1 × 150만원 = 450만원 → 환급 약 90만원
- 기부금 — 50만원 × 15% = 7.5만원
- 연금계좌 — 700만원 × 16.5% = 115.5만원
- **본인 추가 환급 합계 약 213만원**

본인 모두채움 그대로 vs 추가 입력 — 본인 가구 사정 따라 평균 50~250만원 차이.

## 3.5 자주 묻는 질문 (FAQ 5)

**Q1. 본인 부양가족 소득 100만원 이하 — 어떻게 확인?**
A. 본인 부양가족이 직접 본인 소득 확인 (근로·사업·연금·기타). 본인 부양가족도 5월 신고 의무 가능 → 본인이 별도 확인.

**Q2. 본인 12월 30일 IRP 추가 납입 — 신고 반영?**
A. 가능. 본인 금융사 → 1월 자료 갱신 → 모두채움 자동 반영. 본인 직접 확인 권장.

**Q3. 본인 기부금 영수증 분실 — 대응?**
A. 본인 기부 단체에 재발급 신청 (1주 이내). 영수증 없이 신고 X.

**Q4. 본인 부부 합의 안 됨 — 추징 위험?**
A. 본인 + 배우자 동일 자녀 인적공제 시 — 추징 + 가산세. 본인 합의 필수.

**Q5. 본인 ISA 만기 자금 → 연금계좌 이전 — 추가 한도?**
A. 가능 (조건 충족 시). 본인 ISA 5년 보유 후 만기 → 연금계좌 이전 시 추가 한도 1,000만원 인정.

## 3.6 본인 액션 — 4 가지

본 3장 직후 본인 액션:
1. **본인 부양가족 명단** — 4 대 요건 충족 부양가족 정리
2. **본인 기부금 영수증** — 1년 기부 영수증 합산
3. **본인 IRP·연금저축** — 본인 12월 31일까지 본인 한도 (300~900만원) 납입 계획
4. **부부 합의** — 본인 + 배우자 자녀 인적공제 분담 합의
