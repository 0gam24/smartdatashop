## 한 줄 요약

전세사기 = 임대인이 보증금을 반환할 의사·능력 X 임에도 계약 후 보증금 편취.[^1] 본 장은 계약 전 1차 확인 5 단계로 90% 사기 회피.

## 6.1 전세사기 — 5 유형

흔한 유형:[^1]
- **갭 투자 사기** — 임대인이 본인 자금 거의 0 으로 주택 매수, 보증금으로 매수 대금 충당, 가격 하락 시 반환 불능
- **깡통 전세** — 보증금 + 선순위 채권 ≥ 주택가격 → 경매 시 임차인 보증금 회수 불가
- **이중 계약 사기** — 한 주택에 여러 임차인 계약 → 보증금 중복 편취
- **명의 변경 후 잠적** — 계약 후 임대인이 다른 사람에 명의 변경 + 잠적
- **무권 계약** — 임대인이 실제 소유자 아님 (위임장 위조)

## 6.2 1차 확인 5 단계

계약 전 — 본인 직접 확인 5 단계:

**Step 1** — **등기부등본 발급** (필수)
- 대법원 인터넷등기소 (iros.go.kr) — 본인 직접 발급 (수수료 1천원)
- 주의: 임대인이 보여주는 등기부 신뢰 X (위조 가능)

**Step 2** — **선순위 채권 확인** (필수)
- 근저당·압류·임차권 등 확인
- 보증금 + 선순위 채권 ≤ 주택가격의 70~80% (안전 기준)

**Step 3** — **실제 소유자 확인** (필수)
- 등기부의 소유자 = 계약 임대인 (본인 일치)
- 위임장 계약 시 — 위임장 + 인감증명 + 통화 확인

**Step 4** — **주택 가격 확인** (강력 권장)
- 국토교통부 실거래가 (rt.molit.go.kr) — 최근 거래가
- 본인 보증금 vs 주택 가격 = 70~80% 이하 권장

**Step 5** — **HUG 보증 가능 여부** (강력 권장)
- HUG 사전 진단 → 보증 가입 가능 = 안전 신호
- 보증 가입 불가 = 위험 신호

## 6.3 등기부등본 — 무엇을 확인

등기부등본 3 부분:[^2]
- **표제부** — 주택 위치·면적
- **갑구** — 소유권 (현재 소유자 + 매매 이력)
- **을구** — 권리관계 (근저당·임차권·압류 등)

확인 포인트:
- 갑구: 소유자 = 계약 임대인 일치
- 을구: 근저당 합계 → 보증금 + 근저당 ≤ 주택 가격 70~80%

## 6.4 깡통 전세 회피 — 안전 비율

깡통 전세 회피:[^1]
- (보증금 + 선순위 채권) / 주택가격 ≤ **70~80%**
- 80% 초과 → 위험
- 90% 초과 → 매우 위험 (회피 권장)

본인 계산 (예시):
- 보증금 2.5억 + 근저당 0 + 주택 5억 → 비율 50% — 안전 ✅
- 보증금 3억 + 근저당 1억 + 주택 5억 → 비율 80% — 주의 ⚠️
- 보증금 4억 + 근저당 1억 + 주택 5억 → 비율 100% — **회피** ❌

## 6.5 실제 소유자 확인 — 위임장 사기 회피

임대인 본인이 직접 계약 (안전):
- 본인 신분증 + 등기부 소유자 일치
- 즉시 계약 진행 OK

대리인 위임장 계약 (주의):
1. 위임장 + 인감증명 1통
2. 등기부 소유자에 직접 전화 — 위임 확인
3. 위임 사실 통화로 본인 확인
4. 통화 후 계약 진행

위임장 계약 — 본인 통화 안 됨 시 → **즉시 계약 중단**.

## 6.6 임대인 신원 확인 — 추가 확인

임대인이 이상 신호 시:
- 본인 신분증 미제출 → 위험
- 본인 통화 회피 → 위험
- "급매" / "최저가" 강조 → 사기 가능성
- 보증금 송금 즉시 강요 → 사기 가능성

본인 의심 → 계약 중단 + 다른 매물 검토.

## 6.7 계약 후 안전 조치 — 3 단계

계약 직후 본인 액션:[^1]
1. **확정일자 받기** (계약일 또는 다음 날)
   - 동·읍·면 사무소 또는 공증사무소
   - 우선변제권 발생 (경매 시 우선)

2. **전입신고** (이사 직후 24시간 이내)
   - 본인 주민등록상 주소 변경
   - 대항력 발생 (집주인 변경 시 본인 보호)

3. **HUG 보증 가입** (1개월 이내, 5장)
   - 임대인 반환 X 위험 보호
   - 보증료 = 본인 안전 비용

## 6.8 임대 기간 중 — 정기 점검

임대 기간 (보통 2년) 중 정기 확인:
- 매 6개월 — 등기부 재발급 (근저당 추가 여부)
- 임대인 명의 변경 — SMS·이메일 통보 의무
- 본인 우편 임대인 앞 도착 시 — 즉시 본인에 SMS

## 6.9 계약 만료 1개월 전 — 반환 의사 통보

임대차 갱신 X 의향 → 만료 1개월 전 임대인에 통보 의무:[^1]
- 본인 의향 SMS/이메일 + 내용증명 (강력)
- 의사 통보 = 임대인이 새 임차인 모집 + 본인 보증금 반환 준비

통보 후 만료일에 본인 보증금 반환 의무.

## 6.10 본인 피해 발생 — 액션

본인 전세사기 피해 발생 시:[^1]
1. **즉시 신고** — 국토부 전세사기 신고 (molit.go.kr) + 경찰 (112)
2. **HUG 보증 가입 시** — HUG 보증 청구 (5장)
3. **임차권 등기 명령** — 법원에 임차권 등기 → 본인 우선권 보호
4. **소송** — 임대인에 보증금 반환 청구 (변호사·법률구조공단)
5. **전세사기 특별법 활용** — 피해 인정 시 지원금·매입 지원

피해 후 — 본인 자력 회수 평균 1~3년. **사전 1차 확인 이 90% 회피.**

## 6.11 자주 묻는 질문 (FAQ 5)

**Q1. 본인 등기부등본 — 본인 발급?**
A. 본인 인터넷등기소 (iros.go.kr) 본인 발급 (수수료 1천원). 본인 임대인 자료 신뢰 X.

**Q2. 본인 깡통 전세 — 본인 회피?**
A. 본인 (보증금 + 선순위 채권) / 주택가격 ≤ 70~80%.

**Q3. 본인 임대인 위임장 — 본인 안전?**
A. 위임장 + 인감증명 + 본인 임대인 통화 확인. 통화 X → 계약 중단.

**Q4. 본인 피해 발생 — 본인 회수 기간?**
A. 본인 자력 평균 1-3년 (소송). 본인 HUG 보증 시 1-3개월.

**Q5. 본인 의심 매물 — 본인 다음?**
A. 본인 즉시 매물 변경. 본인 의심 = 위험 신호.

## 6.12 본인 액션 — 3 가지

본 6장 직후 본인 액션:
1. **본인 매물별 등기부등본 직접 발급**
2. **본인 1차 확인 5 단계 — 매 계약 전**
3. **본인 확정일자 + 전입신고 + HUG 보증** 3중 보호
