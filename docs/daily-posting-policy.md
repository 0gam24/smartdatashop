# 일일 발행 운영 방침 — 키워드 우선 트래픽 최대화

> 2026-06-02 최초 합의(ADR 0011) → **2026-06-15 키워드 우선·클러스터 집중 전면 개정**(GSC 검색실적 0 진단 후, 운영자 "트래픽 목적 자동 결정" 지시).
> 목적: AdSense "가치 부족" reject 후 재건 국면에서 *신생 사이트가 실제로 이길 수 있는* 자연검색 트래픽을 만드는 운영 모델.
> 위계: `docs/PURPOSE.md` > ADR > `CLAUDE.md` > 본 방침. CLAUDE.md "AdSense 거절 후 가치 재건 정책"과 정합.

## 자동 결정 원칙 (2026-06-15 운영자 영구 지시)

> **앞으로 모든 토픽·전략 결정은 7+ 에이전트 합의로 에이전트(나)가 자동 판단하며, 목적함수는 "트래픽 최대화"다. 운영자에게 선택지를 묻지 않는다.** 발행 자체는 "오늘 포스팅" 트리거로 실행하되, *무엇을 어떤 제목·각도로 쓸지*는 본 방침에 따라 자동 선정한다.

## 핵심 통찰 (2026-06-15 개정 — 검색 실적 진단 후)

신생·저권위 사이트에서 **트래픽 최대화 ≠ 고볼륨 키워드 타겟팅**이다. = **이길 수 있는(winnable) 키워드를 실제로 이기는 것**이다. GSC 진단(색인 1개월차, 노출 66/7일, 클릭 0)이 확인한 사실:

1. **도메인 나이가 1순위 제약** — YMYL(세금·건강·금융)은 권위가 쌓이기 전 거의 노출 안 됨(샌드박스). 1개월차 노출 66은 정상. 전략을 바꿔도 트래픽은 *수개월 뒤* 나온다 — 기대치를 이 타임라인에 맞춘다.
2. **head 키워드는 구조적 패배** — `양도세 비과세` 1페이지는 삼쩜삼·대형 매체 독점. 신생이 못 이긴다.
3. **그래서 롱테일 우선** — `일시적 2주택 양도세 처분기한 2026`처럼 길고 구체적인 질문. 검색량 작아도 1페이지 가능 → 클릭 누적 → 권위 → 점진 상향.

트래픽 엔진의 4축: ① **winnable 롱테일**(검색량 있고 경쟁 약함) ② **토픽 클러스터 집중**(권위 결집) ③ **Google Discover**(순위·권위 불필요한 별도 트랙) ④ **GSC 피드백 루프**(실측 쿼리 → 후속 글).

## 결정 (7+ 에이전트 합의)

### 1. 발행량 — 기본 매일 3편, 상한 5편 (2026-06-12 운영자 결정, 기존 1편/일 폐기)
절대 제약은 편수가 아니라 **글의 질**이다. 구글 scaled content abuse 의 실제 변수는 페이지당 가치이며(제재 사례 = 일 10~500편 무검수 양산), 깊이 있는 글이라면 3편/일은 신뢰·크롤 빈도를 오히려 누적한다.

- 기본 **3편/일**, 상한 **5편/일**
- 모든 글은 골든 레퍼런스 깊이(3,000자+) + ADR 0006 100% + 원본 가치 의무 — **미달 글은 1편도 발행 금지**
- 깊이를 채울 토픽·시간이 부족한 날은 편수를 줄인다 (3편을 채우려 질을 낮추는 것이 최악)

### 2. 토픽 클러스터 — 2개 집중 (2026-06-15 합의 확정)

흩뿌리기 중단. 권위를 결집할 클러스터 2개에 신규 생산을 집중한다.

- **🅰 주력 — 정부 지원금·복지·신청** (지원금·수당·바우처·환급·자격·신청방법·청년정책). *검색의도 강·경쟁 상대적 약·우리 1차출처 파이프라인과 정합·시즌 폭증.* 신생 사이트 최적 니치.
- **🅱 보조 — 생활경제·시의성** (금리·부동산·전기요금·자동차세·세금 시즌). *Discover·실시간 검색 강.* 단기 유입 + 시의성 트래픽.
- **세금(절세·양도세 등)** = 연말정산(1~2월)·종소세(5월) **시즌 기회성**만. 평시 head 키워드 경쟁은 피한다.
- 클러스터 밖 토픽(정부 행사·MOU·인사·단순 속보)은 **생산 중단**. 기존 글은 유지하되 신규는 만들지 않는다.

각 클러스터는 **pillar(대표 정의·총정리 글) + spoke(세부 롱테일 글)** 구조로 쌓고 내부링크로 묶어 토픽 권위를 만든다.

### 2-1. 토픽 선정 방법론 — 키워드 데이터 기반 (6축 점수표 폐기, 2026-06-15)

> 주관 채점(구 6축)을 폐기하고 **실제 검색 데이터**로 선정한다. 핵심은 "수요는 있는데 우리가 이길 수 있는" 롱테일 찾기.

**선정 절차 (매 발행 자동 수행):**
1. **수요 확인** — 후보 키워드를 ① Google 자동완성/연관검색어 ② 네이버 검색광고 키워드도구(무료, 월 검색수 제공) ③ GSC 노출 쿼리로 실수요를 확인. 자동완성에 뜨면 = 사람이 찾는 질문.
2. **롱테일화** — head(`양도세 비과세`)를 구체 질문(`1세대 1주택 일시적 2주택 비과세 처분기한`)으로 좁힌다. 3~5어절·"조건/신청방법/얼마/언제/요건/계산" 결합.
3. **경쟁 점검** — 그 쿼리를 실제 검색해 1페이지가 대형 매체·정부 독점이면 회피, 블로그·중소 사이트가 섞여 있으면 **진입 가능 = 채택**.
4. **클러스터 적합** — §2의 🅰/🅱 안에 드는지 확인. 밖이면 탈락.
5. **검증(ADR 0006)** — 1차 출처로 수치 검증 불가면 무조건 탈락.

**winnability 우선 규칙**: 검색량이 커도 1페이지가 강자 독점이면 *버린다*. 검색량이 작아도 1페이지가 비어 있으면 *집는다*. 작은 승리의 누적이 권위가 되고, 권위가 쌓인 뒤 head로 올라간다.

### 2-2. Google Discover 트랙 (순위 불필요·최단 경로)

Discover는 검색 순위·도메인 권위 없이도 신선·시의성·흥미만으로 노출된다. 신생 사이트의 *가장 빠른* 유입 경로다.
- 🅱 클러스터의 시의성 글(오늘~이번주 발표·시행·마감)을 Discover 적격으로 작성.
- 뉴스사이트맵 자동 포함(이미 가동). 제목은 낚시 금지·구체적, 본문 첫 화면에 핵심.
- SEO(롱테일·축적)와 Discover(시의성·단기)를 **병행**한다.

### 2-3. GSC 피드백 루프 (실측 기반 반복)

GSC에 노출되기 시작한 쿼리가 곧 "우리가 이길 수 있다고 구글이 인정한" 키워드다.
- 매주 GSC 노출 쿼리 점검 → 그 쿼리·연관 롱테일로 후속 spoke 글 생산.
- 노출은 되는데 순위 낮은 글 → 제목·첫 문단·FAQ 보강(재작성).
- 이 루프가 추측을 실측으로 대체한다.

### 2-4. 시즌 캘린더 선점

검색 폭증 시점 **직전(D-7~D-day)**에 발행해 선점.
- **1~2월** 연말정산 · **5월~6월 초** 종소세 · **수시** 정부 지원금·소비쿠폰 공고, 청약, 4대보험·건보료 정산, 전기·가스요금 조정, "상·하반기 달라지는 제도".

### 3. 주 1회 심층 인사이트
그 주의 시의성 토픽들을 교차 종합한 오리지널 분석 1편. 롱테일·LLM 인용 자산 + 원본 가치 앵커.

### 4. 매 글 SEO·AEO·GEO 의무
- **제목** = 실제 검색어를 앞 60자에 배치
- **tldr** = 즉답형 meta description (검색 의도에 직접 답)
- **첫 문단** = 5W1H 즉답 (AEO)
- **구조** = NewsArticle + FAQPage + HowTo LD 자동 트리거 (Q&A 2+, 번호 체크리스트 3+)
- **GEO** = 1차 출처 deep link + 모든 수치 `[^N]` 페어링
- **내부링크** 1~2 (토픽 클러스터)
- **문체** = 독자친화 완전 문장(요약체 금지) + 표 가독성 룰(셀 ≤40자·화살표 0·표 칸 각주 0)

### 5. 주간 리뷰 (운영자 5분)
GSC 유입 쿼리 점검 → 다음 주 토픽·제목 조정(§2-3 루프). 실제 들어오는 검색어에 맞춰 후속 spoke 글을 기획한다.

### 6. 절대 금지
원본 가치 미달 글 발행(편수 채우기), 클러스터 밖 무수요 토픽 신규 생산, head 키워드 무모한 정면승부, 템플릿 양산, 네트워크/자매 링크, 미검증 수치, 요약체.

### 7. 현실적 KPI (기대치 — 매 점검 시 이 타임라인 기준)
- **1개월차(현재)**: 색인·노출 시작 = 정상. 트래픽 0 정상. 조급함 금물.
- **3개월차**: 롱테일 1페이지 진입 시작, 일 클릭 한 자릿수~수십.
- **6개월차**: 클러스터 권위 형성, 일 100+ 가능(니치 집중·일관성 전제).
- **12개월차**: head 키워드 도전 + AdSense 재승인 트래픽 기반.
