## 한 줄 요약

신고 후 세금을 **더 내야 함을 발견** 하면 수정신고 (국세기본법 §45), **더 받아야 함을 발견** 하면 경정청구 (§45의2) 로 대응[^2]. 경정청구는 **법정신고기한 후 5년 이내** 가능하며, 처리 결과는 청구서 접수 후 약 **2개월 이내** 통지[^1].

## 9.0 수정신고 vs 경정청구 — 한눈에 비교

| 구분 | 수정신고 | 경정청구 |
|---|---|---|
| 근거 법령 | 국세기본법 §45 | 국세기본법 §45의2 |
| 발동 조건 | 세금을 **더 내야** 할 때 | 세금을 **더 받아야** 할 때 |
| 가능 기간 | 결정·경정 통지 전까지 | 법정신고기한 후 **5년 이내** |
| 결과 | 추가 산출세액 + 가산세 납부 | 환급 결정 → 본인 계좌 입금 |
| 처리 기간 | 신고 즉시 반영 | 접수 후 **2개월 이내** 통지[^1] |
| 거부 시 | (해당 없음 — 본인 신고) | 행정심판 / 행정소송 가능 |
| 자기시정 감면 | §48의2 — 발견 시점 빠를수록 유리 | (해당 없음 — 본인이 더 받을 권리) |

## 9.1 수정신고 — 본인이 더 낼 게 있을 때

### 사용 시점
- 누락된 사업소득·기타소득 발견
- 부양가족 인적공제 중복 발견 (이중 등록)
- 모두채움 default 가 잘못된 경우

### 절차
홈택스 [신고/납부] → [수정신고] → 기존 신고서 불러와 수정 → 추가 산출세액 + 가산세 납부.

### 자기시정 감면율
국세기본법 §48의2 의 자기시정 감면 (가산세 일부 감면) 은 발견 시점이 빠를수록 유리하다 — 정확한 단계는 법령 직접 확인 권장 (개정 빈번).

## 9.2 경정청구 — 본인이 더 받을 게 있을 때

### 사용 시점
- 인적공제 부양가족 누락
- 기부금 영수증 추가 발견
- 연금계좌 납입액 누락
- 사업 경비 누락

### 가능 기간 — 5년
법정신고기한 다음날부터 **5년 이내** 청구 가능[^1]. 2026년 5월 종합소득세 신고분 → 2031년 5월 31일까지 경정청구 가능.

### 처리 기간
청구서 접수일로부터 **2개월 이내** 세무서 통지·환급[^1].

### 절차
홈택스 [세금신고] → [종합소득세 신고] → [경정청구] → 청구 사유 + 증빙 첨부 → 제출.

### 처리 결과 3 가지
- **수용** — 환급 결정, 본인 계좌 입금
- **부분 수용** — 일부만 인정
- **불수용** — 사유 통지 (본인이 재청구 또는 행정심판)

## 9.3 5년 소멸시효 주의

경정청구 권리도 환급금 미수령도 모두 **5년 소멸시효** (국세기본법 §54 — 별도 검증 권장). 2021년 5월 신고분의 경정청구는 2026년 5월 31일까지가 마지막. 본인이 직접 권리 행사하지 않으면 영구 소멸.

## 9.4 자주 발견되는 누락 — 5 패턴

1. **기부금 영수증 1~2건 누락** — 종교단체 / 정치자금 / 학교 발전기금
2. **부양가족 1명 누락** — 군 복무 자녀 / 분가한 부모
3. **노란우산 납입액 누락** — 가입 첫 해
4. **연금저축 / IRP 신규 가입자 납입** — 12월에 일시 납입
5. **의료비 영수증 합산 누락** — 본인·부양가족

## 9.5 경정청구 vs 행정심판

세무서가 경정청구를 거부한 경우 행정심판 (국세청 조세심판원) 또는 행정소송 진행 가능. 단 비용·시간 대비 환급 예상액 비교 필수 — 1인사업자 기준 환급 200만원 미만이면 비현실적.

## 9.6 검증 권장 항목 (운영자 직접 확인)

- **자기시정 감면율 단계** — 국세기본법 §48의2 본문. 1개월 / 3개월 / 6개월 / 1년 등 시한별 % 는 개정 빈번 → 현행 법령 직접 확인.
- **경정청구 거부 후 행정심판 절차** — 국세청 조세심판원 공식 안내 직접 확인.
- **5년 소멸시효 정확한 시작점** — 법정신고기한 다음날 vs 신고일 다음날 등 디테일.

## 9.7 본인 정정 시나리오 3 종

**시나리오 A — 본인 신고 1주 후 오류 발견**:
- 본인 5월 7일 신고 → 5월 14일 부양가족 누락 발견
- 본인 즉시 경정청구 (홈택스)
- 본인 2개월 이내 환급 결정 → 7월 중순 환급

**시나리오 B — 본인 3년 전 신고 누락 발견**:
- 본인 2023년 5월 신고 → 2026년 5월 종교 기부 50만원 누락 발견
- 본인 경정청구 (3년 경과 — 5년 이내 OK)
- 본인 환급 약 7.5만원 + 처리 약 2개월

**시나리오 C — 본인 추가 납부 의무 발견**:
- 본인 사업소득 누락 5,000만원 발견
- 본인 수정신고 + 가산세 (자기시정 감면 적용)
- 본인 발견 시점 빠를수록 감면 ↑

## 9.8 본인 경정청구 — 본인 5 단계

본인 경정청구 5 단계:
1. **본인 누락·오류 확인** — 본인 자료 점검
2. **본인 증빙 수집** — 영수증·계약서·증명서
3. **홈택스 경정청구** — [신고/납부] → [경정청구]
4. **본인 청구서 + 증빙 첨부** — 본인 사유 명시
5. **본인 결과 통지 대기** — 2개월 이내

## 9.9 자주 묻는 질문 (FAQ 5)

**Q1. 본인 경정청구 — 본인 변호사·세무사 위임?**
A. 본인 직접 가능. 본인 사업 규모 큼 → 세무사 위임 (비용 50~300만원).

**Q2. 본인 경정청구 거부 — 본인 다음 단계?**
A. 본인 행정심판 (조세심판원) → 행정소송. 본인 환급 200만 미만 — 비용 대비 X.

**Q3. 본인 5년 — 정확한 시작점?**
A. 본인 법정신고기한 다음날 (예: 2026년 5월 31일 → 2026년 6월 1일부터 5년).

**Q4. 본인 경정청구 - 본인 환급 입금?**
A. 결정 후 1~4주 본인 계좌 입금. 본인 계좌 변경 시 → 본인 미수령 조회.

**Q5. 본인 매년 경정청구 — 본인 한도?**
A. 한도 X. 본인 매년 누락 발견 시 → 본인 매년 경정청구 가능.
