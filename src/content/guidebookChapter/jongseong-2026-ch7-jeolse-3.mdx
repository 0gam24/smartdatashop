## 한 줄 요약

1인사업자가 5월 신고 직전이라도 즉시 활용 가능한 3대 절세 도구 — (1) 청년창업중소기업 세액감면[^1] (2) 노란우산공제 소득공제[^2] (3) 사업용 신용카드 홈택스 등록[^3]. 이 3가지만 정확히 챙겨도 1인사업자 평균 환급폭이 가시적으로 변한다.

## 7.1 청년창업중소기업 세액감면 (조특법 §6)

### 핵심
만 15세 이상 34세 이하 청년이 창업한 중소기업에 대해 **5년간 산출세액의 일정 비율 세액감면**[^1].

### 감면율 (지역 + 연령 조합)
- **수도권 과밀억제권역 외 청년 (만 34세 이하)** — 100% 감면
- **수도권 과밀억제권역 청년** 또는 **비청년 비수도권 창업** — 50% 감면
- **수도권 비청년 창업** — 25% 감면

### 청년 연령 산정
만 34세까지. 단 군 복무 기간(최대 6년) 차감.

### 적용 업종
조특법 §6 ③ 에 열거된 창업 적용 업종만. 음식점·도소매업·제조업 등 다수 포함되지만 특정 업종 (교습소·대부업 등) 제외 가능. 시행령 별표 직접 확인 필수.

## 7.2 노란우산공제 소득공제

### 핵심
중소기업중앙회가 운영하는 소상공인·사업자 공제. 가입 시 **사업소득에서 직접 소득공제**[^2].

### 공제 한도 (2025년 기준)
- 사업소득 **4,000만원 이하** — 연 500만원
- 사업소득 **4,000만원 초과 ~ 1억원 이하** — 연 300만원
- 사업소득 **1억원 초과** — 연 200만원

(2025년 부터 최대 한도 600만원으로 상향, 법인대표자 총급여 8천만원 이하 요건 — 직접 검증 권장)

### 가입 절차
1. 중소기업중앙회 노란우산공제 사이트 접속 (yumam.kbiz.or.kr)
2. 사업자등록증으로 본인 인증
3. 월 납입액 결정 (5만~100만원)
4. 자동이체 등록

연중 가입해도 그해 납입액 전액 소득공제 적용.

## 7.3 사업용 신용카드 홈택스 등록

6장에서 다룬 사업용 신용카드 등록[^3] 은 **자동 누적** 효과가 핵심이다:
- 홈택스 등록 후 **카드사가 분기마다 국세청에 자동 통보**
- 본인은 종소세 신고 시 별도 입력 없이 신용카드매출전표 합계 자동 반영
- 한도 50매까지 등록 가능

미등록 시: 본인이 종이 영수증을 분기마다 누적 정리해야 함 → 누락 위험 + 시간 낭비.

## 7.4 절세 효과 합산 예시

가상의 1인사업자 A (만 30세, 비수도권 창업, 사업소득 4,000만원, 산출세액 약 400만원) 가정:

| 도구 | 절세 효과 |
|---|---|
| 청년창업 세액감면 100% | -400만원 (산출세액 전액 감면) |
| 노란우산 연 500만원 납입 (4,000만원 이하 한도) | 소득공제 효과 — 한계세율 16.5% 가정 시 약 -82만원 |
| 사업용 카드 등록 (자동 누적) | 직접 절세 효과 X, 시간 절감 + 누락 방지 |

현실 절세액은 본인 산출세액·한계세율·노란우산 납입액에 따라 변동.

## 7.5 검증 권장 항목 (운영자 직접 확인)

- **청년창업 지역구분 100% / 50% / 25% 비율 변경 이력** — 조특법 §6 본문 또는 기재부 보도자료 직접 확인.
- **청년창업 적용 업종 별표** — 시행령 별표 그대로 인용, 임의 요약 금지.
- **노란우산 최대 한도 600만원 (2025년 상향)** — 중소기업중앙회 또는 기재부 공식 안내 확인.

## 7.6 본인 3 도구 활용 시나리오 3 종

**시나리오 A — 본인 만 28세 비수도권 청년 창업**:
- 청년창업 세액감면 **100%** (산출세액 전액 감면, 5년)
- 노란우산 연 500만원 (4천만 이하) → 추가 절세
- 본인 5년 동안 산출세액 0 + 노란우산 누적

**시나리오 B — 본인 만 40세 수도권 비청년 창업**:
- 청년창업 감면 **25%** (수도권 비청년)
- 노란우산 연 300만원 (4천 초과~1억)
- 사업용 카드 등록 → 자동 누적

**시나리오 C — 본인 만 50세 1인사업자 (창업 5년 초과)**:
- 청년창업 감면 자격 X (5년 경과 또는 비청년)
- 노란우산 연 200만원 (1억 초과)
- 사업용 카드 + IRP·연금저축 (3장) 활용

## 7.7 본인 노란우산 — 월 납입액 결정

본인 노란우산 월 납입액 매트릭스:

| 본인 사업소득 | 본인 한도 | 본인 권장 월 납입 |
|---|---|---|
| 4,000만원 이하 | 500만원/년 | 41.6만원 |
| 4,000만~1억 | 300만원/년 | 25만원 |
| 1억 초과 | 200만원/년 | 16.6만원 |

본인 자금 부담 시 — 한도의 50~70% 납입도 OK. 본인 형편 따라 유연.

## 7.8 자주 묻는 질문 (FAQ 5)

**Q1. 본인 청년창업 — 본인 5년 후?**
A. 5년 경과 → 감면 X. 본인 절세 다른 도구 (노란우산·IRP·연금저축) 활용.

**Q2. 본인 노란우산 — 본인 중도 해지?**
A. 가능. 단 본인 해지 시 — 본인 그동안 받은 소득공제 환수 + 본인 일시금에 16.5% 분리 과세.

**Q3. 본인 사업용 카드 등록 — 본인 한도?**
A. 50매. 본인 1 장 시작 → 본인 사업 확장 따라 5-10 매까지.

**Q4. 본인 청년창업 + 노란우산 + IRP — 본인 합산 효과?**
A. 본인 사업소득 4,000만원 + 청년창업 100% + 노란우산 500만 + IRP 700만 → 본인 산출세액 0 + 노란우산·IRP 누적.

**Q5. 본인 비청년이지만 — 본인 절세 도구?**
A. 노란우산 + IRP·연금저축 (3장) + 사업용 카드 + 본인 필요경비 (6장).

## 7.9 본인 액션 — 4 가지

본 7장 직후 본인 액션:
1. **본인 청년창업 자격 확인** — 만 34세 이하 + 5년 이내
2. **본인 노란우산 가입** — 본인 사업자 등록증으로 즉시 가입
3. **본인 사업용 카드 등록** — 홈택스 1주 이내
4. **본인 월 납입액 결정** — 한도의 70~100%
