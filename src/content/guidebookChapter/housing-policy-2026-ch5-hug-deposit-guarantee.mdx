## 한 줄 요약

HUG 전세보증금 반환보증 = 임대인이 보증금 반환 X 시 → HUG 가 대신 지급.[^1] 본 장은 가입 조건 + 보증료 + 청구 절차.

## 5.1 전세보증금 반환보증 — 무엇

전세 계약 = 임차인이 보증금 (수천만~수억) 임대인에 예치.

문제 — 계약 만료 후 임대인이 보증금 반환 X 시:
- 임차인 → 본인 자금 묶임
- 강제 집행 (소송) → 6개월~2년 소요

해결 — HUG 전세보증금 반환보증:[^1]
- 임차인이 HUG 에 보증 가입
- 임대인 반환 X → HUG 가 임차인에 대신 지급
- HUG 가 임대인에 구상권 행사

## 5.2 가입 자격

가입 자격:[^1]
- 본인 = 임차인 (전세 계약자)
- **보증금 + 선순위 채권** ≤ 주택가격의 100%
  - (선순위 채권 = 근저당·압류·다른 보증금)
- 잔여 임대 기간 1년 이상
- 임대차 계약서 + 확정일자 보유

가입 거부:
- 임대인 = HUG 대위변제 채무자 (이전 사고)
- 주택 = 경매·압류 진행 중

## 5.3 가입 절차

**Step 1** — 전세 계약 직후 (1개월 이내 권장):
1. HUG 홈페이지 (khug.or.kr) 또는 가까운 지사 방문
2. 임대차 계약서 + 본인 신분증 + 확정일자 영수증

**Step 2** — 가입 심사 (3~7 영업일):
- HUG 가 주택 가치 + 선순위 채권 확인
- 보증 가능 여부 결정

**Step 3** — 보증료 납부 + 가입 완료:
- 보증료 일시불 (전체 임대 기간)
- 가입 증서 발급

## 5.4 보증료

보증료 = 보증금 × 보증료율 × 임대 기간:[^1]

본인 보증료율:
- 일반 임차인 — **연 0.115~0.154%**
- 청년·신혼부부 — **연 0.0285~0.115%** (할인)
- 저소득층 — 추가 할인

**예시**:
- 보증금 1억 + 일반 + 2년 임대 → 보증료 약 23~30만원 (일시불)
- 보증금 3억 + 일반 + 2년 임대 → 보증료 약 70~92만원
- 보증금 1억 + 청년 우대 + 2년 임대 → 보증료 약 5.7~23만원

## 5.5 보증 한도

HUG 보증 한도:[^2]
- **수도권** — 보증금 7억원 이하
- **수도권 외** — 보증금 5억원 이하

7억 / 5억 초과 시 → HF (한국주택금융공사) 보증 별도 검토.

## 5.6 보증 청구 — 임대인 반환 X 시

임대 만료 후 임대인 반환 X 시 청구 절차:
1. 임대 만료 1개월 이내 임대인에 반환 요구 (내용증명)
2. 임대 만료 + 1개월 후 → HUG 에 보증 사고 신고
3. HUG 가 임대인에 변제 요청 (1차)
4. 임대인 미응답 → HUG 가 임차인에 보증금 대지급 (1~3개월)
5. 이후 HUG 가 임대인에 구상권 행사

본인 자금 확보 — 보증 가입 후 최대 4개월 지연 → 자금 계획 필요.

## 5.7 가입 시점 — 빠를수록 좋음

전세 계약 직후 1개월 내 가입 권장:
- 시간 경과 → 임대인 변경 사고 가능
- 본인 모르는 사이 근저당·압류 → 가입 거부 가능
- 본인 자금 보호 → 최단 시간 가입

가입 기간 — 임대차 잔여 기간 1년 이상이면 OK.

## 5.8 보증 가입 비용 — 본인 부담 vs 임대인 부담

원칙 — 임차인 (본인) 부담.

협의 가능:
- 임대인이 보증료 일부 부담 (특히 새 임대인)
- 계약서에 명시 권장

본인 부담 시 — 보증료 = 본인 안전 비용 (전세 금액 0.1~0.2% 수준).

## 5.9 보증 vs 기타 안전 장치

전세 안전 3 단계:
1. **확정일자** (모든 임차인 필수) — 우선변제권 (경매 시 우선)
2. **전입신고** (모든 임차인 필수) — 대항력 (집주인 변경 시 보호)
3. **HUG 보증** (선택, 강력 권장) — 임대인 반환 X 시 대지급

본인 보증금 1억+ → 본인 보호 위해 3 단계 모두 권장.

## 5.10 보증 가입 못 한 본인 — 무엇

이미 전세 계약 + 보증 미가입 본인 액션:
1. 즉시 HUG 가입 신청 (잔여 1년+ 시)
2. 가입 거부 시 → 본인 계약 만료 + 6개월 전 임대인에 반환 의사 통보
3. 의사 통보 후 임대인 반환 X → 임차권 등기 명령 (법원)
4. 임차권 등기 → 본인 우선 순위 보호

전세사기 회피 1차 확인 (6장) 도 함께 권장.

## 5.11 자주 묻는 질문 (FAQ 5)

**Q1. 본인 HUG 보증 가입 거부 — 본인 의미?**
A. 본인 주택 위험 → 즉시 다른 매물 검토. 본인 거주 위험 신호.

**Q2. 본인 보증료 — 본인 vs 임대인 협의?**
A. 협의 가능. 계약서에 명시. 본인 보증료 = 본인 안전 비용.

**Q3. 본인 보증 청구 시간 — 본인 자금 부담?**
A. 1-3개월 본인 자금 묶임. 본인 사전 자금 계획.

**Q4. 본인 보증 한도 초과 — 본인 대응?**
A. 본인 7억 (수도권) 초과 → HF 보증 별도 검토. 본인 보증금 일부 보호.

**Q5. 본인 청년 우대 — 본인 자격?**
A. 만 19-34세 + 본인 소득 임계. 본인 보증료 0.0285~0.115%.

## 5.12 본인 액션 — 3 가지

본 5장 직후 본인 액션:
1. **본인 전세 계약 직후 HUG 가입** (1개월 이내)
2. **본인 확정일자 + 전입신고** (모든 임차인 의무)
3. **본인 보증료 + 계약서 보관**
