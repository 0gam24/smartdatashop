# `src/lib/format.ts` 수동 검증 표

본 문서는 `format.ts` 각 함수의 기대 출력을 표로 정리한 것이다.
회귀가 의심될 때 이 표의 입력으로 함수를 호출해 결과를 비교한다.

모든 시간 함수는 `Asia/Seoul` 기준이며, 테스트의 `now`는 `2026-05-05T07:30:00+09:00`로 고정한다.
마이너스 부호는 명시된 셀에서 ASCII `-` 가 아닌 유니코드 minus(U+2212, `−`)이다.

---

## formatNumber

| 입력 | 결과 |
|---|---|
| `1234` | `"1,234"` |
| `12345.67` | `"12,345.67"` |
| `0` | `"0"` |
| `-1234` | `"-1,234"` |
| `1234, { fractionDigits: 2 }` | `"1,234.00"` |
| `1.2345, { fractionDigits: 2 }` | `"1.23"` |
| `NaN` | `"NaN"` |
| `Infinity` | `"∞"` |

## formatKoreanNumber

| 입력 | 결과 |
|---|---|
| `1234` | `"1,234"` |
| `10000` | `"1만"` |
| `12345` | `"1만 2,345"` |
| `100000` | `"10만"` |
| `12345678` | `"1,234만 5,678"` |
| `100000000` | `"1억"` |
| `123000000` | `"1억 2,300만"` |
| `1234567890123` | `"1조 2,345억 6,789만 123"` |
| `0` | `"0"` |
| `-12345` | `"−1만 2,345"` |
| `12345, { compact: true }` | `"1만+"` |
| `100000, { compact: true }` | `"10만"` |
| `12345678, { compact: true }` | `"1,234만+"` |

## formatKRW

| 입력 | 결과 |
|---|---|
| `1234` | `"1,234원"` |
| `12345` | `"1만 2,345원"` |
| `12345678` | `"1,234만 5,678원"` |
| `0` | `"0원"` |

## formatDelta

| 입력 | 결과 |
|---|---|
| `5` | `{ sign: "+", value: "5", cls: "delta-up" }` |
| `-5` | `{ sign: "−", value: "5", cls: "delta-down" }` |
| `0` | `{ sign: "", value: "0", cls: "delta-flat" }` |
| `-24.6` | `{ sign: "−", value: "24.6", cls: "delta-down" }` |
| `0, { showZeroSign: true }` | `{ sign: "+", value: "0", cls: "delta-flat" }` |
| `NaN` | `{ sign: "", value: "NaN", cls: "delta-flat" }` |

## formatRelativeTime

`now = "2026-05-05T07:30:00+09:00"` 기준.

| 입력 | 결과 |
|---|---|
| `"2026-05-05T07:00:00+09:00"` | `"30분 전"` |
| `"2026-05-05T05:00:00+09:00"` | `"2시간 전"` |
| `"2026-05-04T07:00:00+09:00"` | `"어제"` |
| `"2026-05-02T07:00:00+09:00"` | `"3일 전"` |
| `"2026-04-25T07:00:00+09:00"` | `"4월 25일"` |
| `"2025-04-25T07:00:00+09:00"` | `"2025년 4월 25일"` |
| `"2026-05-06T07:00:00+09:00"` | `"내일"` |
| `"2026-05-05T07:30:00+09:00"` | `"방금 전"` |
