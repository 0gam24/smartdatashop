## 한 줄 요약

필요경비는 사업 운영에 든 비용으로 — 영수증 보관 5년 의무.[^1] 매입·임대료·통신·교통·접대비 등 분류별 기준이 다르며, 영수증 형태 (세금계산서·신용카드·현금영수증) 도 인정 여부에 영향.

## 5.1 필요경비 인정 기본 원칙

[^1]:
- **사업 관련성** — 사업 운영에 필요한 지출
- **금액 합리성** — 사업 규모 대비 과도하지 않음
- **증빙 자료** — 세금계산서·신용카드영수증·현금영수증·간이영수증
- **5년간 보관 의무** — 세무조사 시 제출 가능해야 함

## 5.2 주요 필요경비 분류

| 분류 | 인정 여부 | 비고 |
|---|---|---|
| 사업장 임대료 | ✓ | 세금계산서 필수 |
| 사업용 통신비 (휴대폰·인터넷) | ✓ | 사업 비중 분리 시 일부 |
| 사업용 차량 유지비 | ✓ (일부) | 차종·용도 제한 |
| 사업용 교통비 (택시·KTX 등) | ✓ | 영수증 필수 |
| 사업장 공과금 (전기·가스·수도) | ✓ | 사업장만 |
| 접대비 | ✓ (한도 내) | 매년 기본 한도 + 매출별 추가 |
| 광고선전비 | ✓ | 세금계산서·영수증 |
| 사업용 사무용품·소모품 | ✓ | 영수증 |
| 회계·세무·법무 컨설팅 | ✓ | 세금계산서 |
| 직원 인건비 | ✓ | 1인사업자는 본인 인건비 X |
| 본인 식대 | X | 사업 관련 식대만 (접대비) |
| 본인 의료비·생활비 | X | 사업 무관 |

## 5.3 접대비 한도

[^2] 기준:
- **기본 한도** — 매년 일정 금액 (보통 1,200만원 또는 3,600만원 등 변동, 정책 연도별 확인)
- **수입금액 한도** — 매출의 일정 비율 (0.05~0.3%, 매출별 차등)
- 두 한도 중 큰 금액까지 필요경비 인정

접대비 영수증 1건당 일정 금액 (3만원·5만원) 초과 시 — 신용카드·현금영수증 의무 (영수증 미수령 시 비공제).

## 5.4 사업용 vs 개인용 분리

1인사업자는 사업과 개인 지출이 섞이기 쉽다. 권장 분리 방법:

**1. 사업용 통장·카드 분리 (필수)**
- 사업자등록 후 즉시 사업용 통장 개설
- 모든 사업 거래는 사업용 통장에서 처리
- 사업용 신용카드 사용

**2. 사업용 휴대폰 (선택)**
- 본인 휴대폰을 사업 사용도 가능 (사용 비중에 따라 일부 경비 인정)
- 또는 별도 회선 (전액 경비)

**3. 사업용 차량 (선택)**
- 일반 승용차 (자가용) 는 사업용 인정 제한
- 화물차·승합차는 사업용 인정 폭 넓음

## 5.5 영수증 종류별 인정 가치

[^1] 기준:

| 영수증 종류 | 부가세 매입세액 공제 | 종소세 필요경비 |
|---|---|---|
| 세금계산서 | ✓ 100% | ✓ |
| 신용카드영수증 | ✓ 100% | ✓ |
| 현금영수증 | ✓ 100% | ✓ |
| 간이영수증 (3만원 이하) | △ 일부 | ✓ |
| 영수증 미수령 | X | 부분 인정 (소액·증빙 X) |

매입 시 가능하면 **세금계산서 또는 카드영수증·현금영수증** 수령 — 매입세액 공제와 필요경비 인정 모두 가능.

## 5.6 영수증 보관 방식

5년 보관 의무.[^1] 보관 방식:

**디지털 (권장)**:
- 영수증 사진 촬영 → 클라우드 저장 (Google Drive·Dropbox 등)
- 회계 SaaS 자동 OCR 처리 (9장)
- 검색·정리 효율 ↑

**물리적 보관**:
- 월별 봉투·바인더
- 보관 공간 부담
- 분실·훼손 위험

대부분의 회계 SaaS 는 영수증 디지털 보관·자동 분류 기능 제공.

## 5.7 자주 누락하는 경비

1인사업자가 흔히 누락하는 경비:
- **홈택스·하이패스 등 사업 관련 카드 사용**
- **사업용 도서·교육비** (자기개발은 일부 인정)
- **사업용 보험료** (사업장 화재보험·전문가책임보험)
- **외주 비용** (디자인·번역·사진 외주)
- **소셜미디어·SaaS 구독료** (Slack·Notion·Adobe 등)
- **카드 연회비** (사업용 카드만)

**누락한 경비 = 미공제 = 세금 증가**. 매월 정리 권장.

## 5.8 매월 영수증 정리 5단계

1. **사업용 카드·통장 거래 내역 다운로드** (홈택스·은행 앱)
2. **거래별 사업·개인 분류**
3. **영수증 매칭 (디지털 사진 또는 회계 SaaS)**
4. **분류별 합계 (임대료·통신·교통 등)**
5. **매월 1일 또는 마지막 날 점검**

매월 정리하면 5월 종소세 + 7월·1월 부가세 신고 부담 대폭 감소.

## 5.9 본인 경비 시나리오 3 종

**시나리오 A — 본인 디자이너·개발자**:
- 본인 통신비 70% (사업 사용 비율)
- 본인 노트북·소프트웨어 (사업 자산)
- 본인 사무용품·교육비

**시나리오 B — 본인 점포 운영 (도소매)**:
- 본인 임차료·인건비·매입
- 본인 광고비·교통비
- 본인 직원 4대보험 사업주 부담

**시나리오 C — 본인 출장 영업직**:
- 본인 출장 식대·교통·숙박
- 본인 기업업무추진비 (거래처 식사)
- 본인 차량유지비 (한도)

## 5.10 자주 묻는 질문 (FAQ 5)

**Q1. 본인 가족 식사비 — 본인 경비?**
A. 본인 사업 관련 X → 경비 X. 본인 거래처 식사 → 본인 기업업무추진비 (한도).

**Q2. 본인 3만원 초과 — 본인 증빙?**
A. 적격증빙 4종 (세금계산서·계산서·신용카드·현금영수증) 수취 의무.

**Q3. 본인 차량유지비 — 본인 100%?**
A. 업무전용보험 + 운행기록부 100% → 한도 X. 미가입 → 한도 1,500만원.

**Q4. 본인 통신비 — 본인 비율?**
A. 본인 사업 사용 비율 (예: 70%)만 경비. 100% X.

**Q5. 본인 영수증 분실 — 본인 대응?**
A. 본인 발행 기관 재발급 또는 본인 카드 매출전표 (자동 누적).

## 5.11 본인 액션 — 4 가지

본 5장 직후 본인 액션:
1. **본인 사업용 카드 + 통장** 분리
2. **본인 홈택스 사업용 카드 등록** (자동 누적)
3. **본인 회계 SaaS** 가입 + 매일 영수증 업로드
4. **본인 매월 5분 — 거래 매칭**
