# Google SEO 바이브코딩 가이드 패키지

Google 검색 센터(https://developers.google.com/search/docs)의 공식 가이드를 기반으로
바이브코딩(Vibe Coding)에서 바로 참조·재사용할 수 있도록 정리한 한국어 마크다운 모음입니다.

## 📦 포함 파일
| 파일 | 내용 |
|------|------|
| 01-search-essentials.md | Google 검색 Essentials (기술 요구사항·스팸 정책·핵심 권장사항) |
| 02-seo-fundamentals.md | SEO 기본 가이드 (검색 동작 원리, URL/콘텐츠/링크/이미지 원칙) |
| 03-crawling-indexing.md | 크롤링·색인 (Googlebot, robots.txt, sitemap, 캐싱) |
| 04-ranking-appearance.md | 순위·검색 노출 (title, snippet, JSON-LD, E-E-A-T) |
| 05-monitoring-debugging.md | Search Console·Core Web Vitals·문제 해결 |
| 06-site-specific-guides.md | 사이트 유형별(전자상거래·다국어·SPA·뉴스·동영상) |
| 07-vibe-coding-seo-template.md | HTML/Next.js 즉시 적용 템플릿 + AI 프롬프트 |

## 🚀 사용 방법
1. AI 코딩 어시스턴트(Cursor, Claude Code, Copilot 등)의 컨텍스트/프로젝트 룰에 추가
2. 새 페이지/컴포넌트 작성 시 해당 카테고리 파일을 참고로 첨부
3. 배포 전 각 파일 하단의 "바이브코딩 체크리스트" 항목 확인
